# ✅ 报销管理 - 添加发票字段 - 部署说明

## 📋 功能说明

将报销申请表单从简单的4个字段扩展为完整的发票管理字段，包括：
- 单位名称、发票号码
- 打款日期、支出金额
- 类目、项目
- 摘要（原报销事由）
- 收到发票日期、发票类型
- 开票金额、税率、扣税额
- 不含税金额、税金
- 报销人、开票日期
- 备注

## 🎯 主要特性

### 1. **税金自动计算**
- 输入开票金额和税率
- 自动计算：扣税额、不含税金额、税金
- 计算公式：
  - 税额 = 开票金额 × 税率 / (100 + 税率)
  - 不含税金额 = 开票金额 - 税额

### 2. **字段验证**
必填字段：
- ✅ 单位名称
- ✅ 报销人
- ✅ 支出金额
- ✅ 项目
- ✅ 摘要

## ⚠️ 注意事项

1. **数据兼容性**
   - 旧的报销记录中，新字段将为NULL
   - 不影响现有数据

2. **表单验证**
   - 新增了company_name和project为必填
   - 如果用户没有填写，将提示错误

3. **税金计算**
   - 自动计算是在前端完成的
   - 后端不进行计算，直接存储前端传来的值
   - 如需修改计算公式，修改 `src/views/Reimbursement/index.vue` 中的 `calculateTax()` 函数
