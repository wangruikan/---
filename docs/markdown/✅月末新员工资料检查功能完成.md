# ✅ 月末新员工资料检查功能完成

## 功能概述

实现了月末自动检测新入职员工资料上传情况的定时任务，当发现资料缺失时，自动为业务人员（第一个审批节点）生成考核记录。

## 核心功能

### 1. 自动检测逻辑

```
月末定时任务触发
    ↓
查找当月入职的员工
    ↓
检查每个员工的资料上传情况
    ↓
如果发现资料缺失
    ↓
获取业务人员（第一个审批节点用户）
    ↓
为业务人员生成考核记录
```

### 2. 检查的资料类型

- ✅ **身份证** (id_card)
- ✅ **学历证明** (diploma)  
- ✅ **个人简历** (resume)
- ✅ **证件照片** (photo)
- ✅ **银行卡** (bank_card)
- ✅ **劳动合同** (contract)

### 3. 考核记录生成规则

- **业务类型**: `document_upload` (资料收集)
- **责任人**: 第一个审批节点的用户
- **截止时间**: 员工入职后7个工作日
- **状态**: `pending` (待处理)
- **备注**: 详细列出缺失的资料类型

## 工作流程详解

### 1. 员工筛选

```php
// 查找本月入职的员工
$newEmployees = Employee::where('account_set_id', $accountSet->id)
    ->whereBetween('hire_date', [$startOfMonth, $endOfMonth])
    ->get();
```

### 2. 资料检查

```php
// 检查每种必需资料是否已上传
foreach ($requiredDocuments as $type => $name) {
    $hasDocument = DocumentDelivery::where('account_set_id', $employee->account_set_id)
        ->where('employee_id', $employee->id)
        ->where('document_type', $type)
        ->where('status', 'completed')
        ->exists();
        
    if (!$hasDocument) {
        $missingDocuments[] = $name;
    }
}
```

### 3. 业务人员获取

```php
// 获取第一个审批节点的用户
$firstApprovalRecord = ApprovalRecord::whereHas('instance', function($query) use ($accountSetId) {
        $query->where('account_set_id', $accountSetId);
    })
    ->where('step_order', 1) // 第一个审批节点
    ->orderBy('created_at', 'desc')
    ->first();
```

### 4. 考核记录创建

```php
AssessmentRecord::create([
    'account_set_id' => $accountSetId,
    'business_type' => 'document_upload',
    'business_id' => $employee->id,
    'business_name' => "新员工 {$employee->name} 资料收集",
    'handler_id' => $businessUser->id,
    'handler_name' => $businessUser->name,
    'deadline_date' => $deadlineDate, // 入职后7个工作日
    'status' => 'pending',
    'remark' => "缺失资料: " . implode(', ', $missingDocuments)
]);
```

## 防重复机制

系统会检查是否已存在相同的考核记录，避免重复生成：

```php
$existingRecord = AssessmentRecord::where('account_set_id', $accountSetId)
    ->where('business_type', 'document_upload')
    ->where('business_id', $employee->id)
    ->where('handler_id', $businessUser->id)
    ->whereMonth('created_at', Carbon::now()->month)
    ->whereYear('created_at', Carbon::now()->year)
    ->first();
```

## 配置说明

### 定时任务时间

```php
// 每月最后一天晚上21点执行
$schedule->command('assessment:check-new-employee-documents')
         ->monthlyOn(31, '21:00')
         ->timezone('Asia/Shanghai')
         ->when(function () {
             return Carbon::now()->day === Carbon::now()->endOfMonth()->day;
         });
```

### 截止日期计算

```php
// 入职后7个工作日（不包括周末）
$deadlineDate = Carbon::parse($employee->hire_date)->addWeekdays(7);
```
