# ✅ 添加医保计算逻辑

## 🎯 问题

之前的代码只处理了 `social_security_types`（社保），但**没有处理 `medical_insurance_types`（医保）**！

## 📊 medical_insurance_types JSON 结构

```json
[
  {
    "id": 1,
    "name": "医保1",
    "base_amount": "5000.00",        // ✅ 医保基数（可能在这里）
    "company_ratio": "0.1199",       // ✅ 单位比例 11.99%
    "employee_ratio": "0.7002",      // ✅ 个人比例 70.02%
    "is_enabled": true
  }
]
```

## ✅ 添加医保处理逻辑

**文件：** `app/Http/Controllers/SalaryController.php`  
**位置：** 第 965-1010 行

### 实现逻辑

```php
// 计算医保（单位和个人）- 从 medical_insurance_types JSON 字段读取
if ($insurancePersonnel->medical_insurance_types) {
    $medicalInsuranceTypes = is_string($insurancePersonnel->medical_insurance_types)
        ? json_decode($insurancePersonnel->medical_insurance_types, true)
        : $insurancePersonnel->medical_insurance_types;
    
    if (is_array($medicalInsuranceTypes)) {
        foreach ($medicalInsuranceTypes as $index => $typeConfig) {
            // 医保基数：优先使用 base_amount，否则使用 employee_medical_insurance_base
            $base = floatval($typeConfig['base_amount'] ?? $insurancePersonnel->employee_medical_insurance_base ?? 0);
            $companyRatio = floatval($typeConfig['company_ratio'] ?? 0);
            $employeeRatio = floatval($typeConfig['employee_ratio'] ?? 0);
            
            $companyAmount = $base * $companyRatio;
            $personalAmount = $base * $employeeRatio;
            
            // 加入社保合计
            $companySocialSecurity += $companyAmount;
            $personalSocialSecurity += $personalAmount;
            
            // 保存明细（添加到 social_security 数组中）
            $details['social_security'][] = [
                'name' => $typeConfig['name'] ?? '未知险种',
                'base' => $base,
                'company_ratio' => $companyRatio,
                'company_amount' => round($companyAmount, 2),
                'personal_ratio' => $employeeRatio,
                'personal_amount' => round($personalAmount, 2),
            ];
        }
    }
}
```

### 关键点

1. **基数获取优先级**：
   - 优先使用：`$typeConfig['base_amount']`（JSON中的基数）
   - 其次使用：`$insurancePersonnel->employee_medical_insurance_base`（表字段）

2. **金额计算**：
   ```php
   单位金额 = 基数 × 单位比例
   个人金额 = 基数 × 个人比例
   ```

3. **加入合计**：
   - 医保金额**加入** `$companySocialSecurity` 和 `$personalSocialSecurity`
   - 因为医保是社保的一部分

4. **保存明细**：
   - 添加到 `$details['social_security']` 数组
   - 和其他社保险种一起显示

## 📊 数据流程

```
insurance_personnel 表
    ↓
medical_insurance_types (JSON字段)
    ↓
解析 JSON → 遍历每个医保险种
    ↓
计算：基数 × 比例 = 金额
    ↓
加入社保合计 + 保存明细
    ↓
返回给前端显示
```
