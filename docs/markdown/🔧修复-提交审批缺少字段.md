# 🔧 修复提交审批错误 - 缺少 account_set_id

## ❌ 错误信息

```
{
  "success": false,
  "message": "提交失败：SQLSTATE[HY000]: General error: 1364 Field 'account_set_id' doesn't have a default value (Connection: mysql, SQL: insert into `process_approvals` (`status`, `updated_at`, `created_at`) values (pending, 2025-11-02 22:59:06, 2025-11-02 22:59:06))"
}
```

---

## 🔍 问题原因

在创建 `ProcessApproval` 审批实例时，**缺少必填字段 `account_set_id`**。

### 错误的代码
```php
// 缺少 account_set_id
$approvalInstance = ProcessApproval::create([
    'type' => 'invoice',
    'related_id' => $application->id,
    'status' => 'pending',
    'current_node_id' => 2,
    'applicant_id' => $application->submitter_id,
]);
```

---

## ✅ 解决方案

### 1. `submit()` 方法 - 提交审批

#### 修改前（错误）
```php
DB::beginTransaction();
try {
    // 创建审批实例（跳过第1节点）
    $approvalInstance = ProcessApproval::create([
        'type' => 'invoice',
        'related_id' => $application->id,
        'status' => 'pending',
        'current_node_id' => 2,
        'applicant_id' => $application->submitter_id,
    ]);
    
    // ...
}
```

#### 修改后（正确）
```php
DB::beginTransaction();
try {
    // 创建审批实例（跳过第1节点）
    $approvalInstance = ProcessApproval::create([
        'account_set_id' => $application->account_set_id,  // ← 新增
        'type' => 'invoice',
        'related_id' => $application->id,
        'status' => 'pending',
        'current_node_id' => 2,
        'applicant_id' => $application->submitter_id,
    ]);
    
    // ...
}
```

## 🎯 修改说明

### `process_approvals` 表结构
```sql
CREATE TABLE `process_approvals` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT,
  `account_set_id` bigint unsigned NOT NULL,  -- ✅ 必填字段
  `type` varchar(50) NOT NULL,
  `related_id` bigint unsigned NOT NULL,
  `status` varchar(20) NOT NULL,
  `current_node_id` int DEFAULT NULL,
  `applicant_id` bigint unsigned NOT NULL,
  `created_at` timestamp NULL DEFAULT NULL,
  `updated_at` timestamp NULL DEFAULT NULL,
  PRIMARY KEY (`id`)
);
```

### 为什么需要 `account_set_id`？
1. **多账套支持**：系统支持多个账套，每个审批实例必须关联到特定账套
2. **数据隔离**：不同账套的审批数据需要隔离
3. **权限控制**：审批人只能审批自己账套的申请
