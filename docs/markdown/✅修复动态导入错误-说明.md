# 修复动态导入错误 - 说明文档

## 🐛 问题描述

**错误信息：**
```
Create payment error: TypeError: Failed to fetch dynamically imported module: 
http://localhost:3000/src/api/salaryPaymentRequests.js
```

**发生场景：**
- 在工资管理页面发起付款申请时
- 点击"提交付款申请"按钮后报错

## 🔍 问题原因

代码中使用了动态导入（`import()`）来加载API模块：

```javascript
// ❌ 错误的写法（动态导入）
const { submitSalaryPaymentRequest } = await import('@/api/salaryPaymentRequests')
const { completeSalaryPaymentSubmission } = await import('@/api/salaryPaymentRequests')
const { uploadSalaryPaymentAttachment } = await import('@/api/salaryPaymentRequests')
```

**为什么会失败：**
1. Vite开发服务器在处理动态导入时可能存在模块解析问题
2. 路径别名 `@` 在动态导入中可能无法正确解析
3. 模块缓存和热更新可能导致动态导入失败

## ✅ 解决方案

将动态导入改为静态导入：

### 1. 在文件顶部添加静态导入

**修改位置：** `src/views/Salaries/index.vue` 第735-739行

```javascript
// ✅ 正确的写法（静态导入）
import {
  submitSalaryPaymentRequest,
  completeSalaryPaymentSubmission,
  uploadSalaryPaymentAttachment
} from '@/api/salaryPaymentRequests'
```

### 2. 直接使用已导入的函数

**修改位置：** `src/views/Salaries/index.vue` 第1408-1439行

```javascript
// ✅ 直接使用，无需动态导入
const response = await submitSalaryPaymentRequest({
  salary_approval_id: paymentForm.salary_approval_id,
  remarks: paymentForm.remarks,
  current_account_set_id: accountSetStore.currentAccountSetId
})

// ...

const completeResponse = await completeSalaryPaymentSubmission({
  payment_request_id: paymentRequestId
})
```

**修改位置：** `src/views/Salaries/index.vue` 第1487行

```javascript
// ✅ 直接使用，无需动态导入
await uploadSalaryPaymentAttachment(formData)
```

## 💡 为什么静态导入更好

### 优点

1. **性能更好**
   - 在应用启动时就加载模块
   - 避免运行时的模块解析开销
   - 更好的打包优化

2. **更可靠**
   - 编译时就能发现错误
   - 不受运行时环境影响
   - 避免路径解析问题

3. **代码更清晰**
   - 在文件顶部统一管理依赖
   - 更容易理解模块依赖关系
   - 便于代码维护

### 动态导入的适用场景

动态导入通常只在以下场景使用：

1. **代码分割（Code Splitting）**
   ```javascript
   // 路由懒加载
   const Home = () => import('./views/Home.vue')
   ```

2. **条件加载**
   ```javascript
   // 根据条件加载不同模块
   if (condition) {
     const module = await import('./moduleA')
   } else {
     const module = await import('./moduleB')
   }
   ```

3. **按需加载大型库**
   ```javascript
   // 只在需要时加载大型第三方库
   const chart = await import('chart.js')
   ```
