# ✅ 修复变量作用域问题

## 🐛 问题描述

生成"老年公寓"项目的工资表时报错：
```
Undefined variable $largeMedicalCompanyAmount
at line 1527 in SalaryController.php
```

---

## 🔍 问题原因

### 变量作用域错误

**文件**：`app/Http/Controllers/SalaryController.php`  
**方法**：`calculateMonthlyInsurance()`

**问题代码结构**：
```php
private function calculateMonthlyInsurance($employee, $month)
{
    $companySocialSecurity = 0;
    $personalSocialSecurity = 0;
    $companyHousingFund = 0;
    $personalHousingFund = 0;
    // ❌ 缺少大额医疗变量的初始化
    
    if ($insurancePersonnel) {
        // ... 社保计算 ...
        // ... 公积金计算 ...
        
        // 大额医疗计算
        $largeMedicalCompanyAmount = 0;  // ❌ 只在if块内定义
        $largeMedicalEmployeeAmount = 0;  // ❌ 只在if块内定义
        
        if ($insurancePersonnel->large_medical_insurance_config) {
            // 计算大额医疗
            $largeMedicalCompanyAmount = ...;
            $largeMedicalEmployeeAmount = ...;
        }
    }  // ← if块结束，变量作用域结束
    
    // ❌ 这里使用变量，但变量已经不在作用域内了
    $result = [
        'company_large_medical' => round($largeMedicalCompanyAmount, 2),  // ← 报错
        'personal_large_medical' => round($largeMedicalEmployeeAmount, 2), // ← 报错
    ];
}
```

---

## ✅ 解决方案

### 在方法开头初始化变量

将 `$largeMedicalCompanyAmount` 和 `$largeMedicalEmployeeAmount` 的初始化移到方法开头，与其他变量一起初始化。

**修复后的代码**：
```php
private function calculateMonthlyInsurance($employee, $month)
{
    $companySocialSecurity = 0;
    $personalSocialSecurity = 0;
    $companyHousingFund = 0;
    $personalHousingFund = 0;
    $largeMedicalCompanyAmount = 0;      // ✅ 在方法开头初始化
    $largeMedicalEmployeeAmount = 0;     // ✅ 在方法开头初始化
    
    // ... 详细计算逻辑 ...
    
    if ($insurancePersonnel) {
        // ... 社保计算 ...
        // ... 公积金计算 ...
        
        // 大额医疗计算
        if ($insurancePersonnel->large_medical_insurance_config) {
            // 直接赋值，不重新定义
            $largeMedicalCompanyAmount = ...;
            $largeMedicalEmployeeAmount = ...;
        }
    }
    
    // ✅ 现在可以安全使用这些变量了
    $result = [
        'company_large_medical' => round($largeMedicalCompanyAmount, 2),
        'personal_large_medical' => round($largeMedicalEmployeeAmount, 2),
    ];
}
```

## 🎯 为什么会出现这个问题？

### 变量作用域规则

在PHP中，变量的作用域由花括号 `{}` 定义：

```php
function example() {
    if (condition) {
        $var = 10;  // $var 只在这个if块内有效
    }
    
    echo $var;  // ❌ 错误：$var 未定义
}
```

正确的做法：
```php
function example() {
    $var = 0;  // 在函数开头初始化
    
    if (condition) {
        $var = 10;  // 修改变量值
    }
    
    echo $var;  // ✅ 正确：$var 已定义，值为0或10
}
```

## 📝 学习要点

### 1. 变量初始化的最佳实践

```php
function calculate() {
    // ✅ 好习惯：在方法开头初始化所有变量
    $result1 = 0;
    $result2 = 0;
    $result3 = 0;
    
    // ... 计算逻辑 ...
    
    return [
        'result1' => $result1,
        'result2' => $result2,
        'result3' => $result3,
    ];
}
```

### 2. 避免在条件块内定义变量

```php
// ❌ 不好的做法
if ($condition) {
    $importantVar = calculateValue();
}
return $importantVar;  // 如果$condition为false，这里会报错

// ✅ 好的做法
$importantVar = null;
if ($condition) {
    $importantVar = calculateValue();
}
return $importantVar;  // 安全，最坏情况返回null
```
