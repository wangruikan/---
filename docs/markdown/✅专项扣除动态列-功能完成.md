# ✅ 专项扣除动态列 - 功能完成

## 🎯 功能说明

在工资表中添加**个人所得税专项附加扣除**的动态列，格式与保险数据类似：

1. **第一行**：大标题 "个人所得税专项附加扣除"
2. **第二行**：动态生成的扣除项目列（如 "1212"、"测试扣除项目" 等）
3. **数据行**：每个员工对应项目的扣除金额

## ✅ 后端实现

### 1. 添加 `calculateSpecialDeduction` 方法

```php
/**
 * 计算员工专项扣除明细
 */
private function calculateSpecialDeduction($employee, $month)
{
    // 查询该员工的专项扣除详情
    $deductionDetail = \App\Models\EmployeeDeductionDetail::where('employee_id', $employee->id)
        ->where('is_active', 1)
        ->first();
    
    if (!$deductionDetail || empty($deductionDetail->deduction_items)) {
        return [
            'total' => 0,
            'items' => []
        ];
    }
    
    // 解析扣除项目（格式：1:100|2:100）
    $items = [];
    $total = 0;
    $parts = explode('|', $deductionDetail->deduction_items);
    
    foreach ($parts as $part) {
        if (strpos($part, ':') !== false) {
            list($itemId, $amount) = explode(':', $part);
            $item = \App\Models\SpecialDeductionItem::find($itemId);
            if ($item) {
                $items[] = [
                    'id' => $item->id,
                    'name' => $item->name,
                    'amount' => floatval($amount),
                ];
                $total += floatval($amount);
            }
        }
    }
    
    return [
        'total' => round($total, 2),
        'items' => $items
    ];
}
```

### 2. 获取所有专项扣除项目

```javascript
// 获取所有专项扣除项目（从所有员工的扣除明细中提取）
const getSpecialDeductionItems = (details) => {
  if (!details || details.length === 0) return []
  
  const itemsMap = new Map()
  
  details.forEach(row => {
    if (row.special_deduction_details && row.special_deduction_details.items) {
      row.special_deduction_details.items.forEach(item => {
        if (!itemsMap.has(item.id)) {
          itemsMap.set(item.id, {
            id: item.id,
            name: item.name
          })
        }
      })
    }
  })
  
  return Array.from(itemsMap.values())
}
```

### 3. 获取某个员工某个扣除项目的金额

```javascript
// 获取某个员工某个扣除项目的金额
const getSpecialDeductionAmount = (row, itemId) => {
  if (!row.special_deduction_details || !row.special_deduction_details.items) {
    return '-'
  }
  
  const item = row.special_deduction_details.items.find(i => i.id === itemId)
  if (!item) {
    return '-'
  }
  
  return formatMoney(item.amount)
}
```

## 📊 工作原理

### 1. 数据存储

专项扣除数据存储在 `employee_deduction_details` 表中：

```
employee_deduction_details:
  - employee_id: 16
  - deduction_items: "1:1|2:100|3:100"  ← 格式：项目ID:金额|项目ID:金额
  - total_amount: 201
  - is_active: 1
```

### 2. 后端解析

```php
// 解析 "1:1|2:100|3:100"
$parts = explode('|', $deductionDetail->deduction_items);
// 得到: ["1:1", "2:100", "3:100"]

foreach ($parts as $part) {
    list($itemId, $amount) = explode(':', $part);
    // 1 => 1, 2 => 100, 3 => 100
}
```

### 3. 前端动态列生成

```javascript
// 从所有员工的数据中提取扣除项目
getSpecialDeductionItems(salaryDetails)
// 返回: [{id: 1, name: "1212"}, {id: 2, name: "测试扣除项目"}, ...]

// 然后动态生成列
<el-table-column v-for="item in items" :label="item.name">
```

### 4. 显示员工金额

```javascript
// 查找该员工是否有这个扣除项目
getSpecialDeductionAmount(row, itemId)
// 有 → 返回 "¥100.00"
// 没有 → 返回 "-"
```
