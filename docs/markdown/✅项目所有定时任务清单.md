# 项目所有定时任务清单

## 📋 定时任务总览

项目中共有 **6 个定时任务**，涵盖以下模块：
1. 参保管理
2. 基数管理
3. 资料交付管理

---

## 🕐 定时任务详细清单

### 1️⃣ 检查参保入职超期
**命令：** `assessment:check-insurance-deadlines`
**执行时间：** 每天早上 8:00
**功能：** 检查员工参保入职是否超期，发送提醒通知

```php
$schedule->command('assessment:check-insurance-deadlines')
         ->dailyAt('08:00')
         ->timezone('Asia/Shanghai');
```

**等效 Cron：**
```bash
0 8 * * * cd /your/project && php artisan assessment:check-insurance-deadlines
```

---

### 2️⃣ 生成参保明细记录
**命令：** `insurance:generate-monthly-details`
**执行时间：** 每月最后一天 23:00
**功能：** 自动生成当月的参保明细记录

```php
$schedule->command('insurance:generate-monthly-details')
         ->monthlyOn(31, '23:00')
         ->timezone('Asia/Shanghai')
         ->when(function () {
             return Carbon::now()->day === Carbon::now()->endOfMonth()->day;
         });
```

**等效 Cron：**
```bash
0 23 28-31 * * [ $(date -d tomorrow +\%d) -eq 1 ] && cd /your/project && php artisan insurance:generate-monthly-details
```

---

### 3️⃣ 处理员工调基
**命令：** `base:process-employee-adjustments`
**执行时间：** 每天早上 9:00
**功能：** 检查员工调基是否到达生效时间，自动更新基数

```php
$schedule->command('base:process-employee-adjustments')
         ->dailyAt('09:00')
         ->timezone('Asia/Shanghai');
```

**等效 Cron：**
```bash
0 9 * * * cd /your/project && php artisan base:process-employee-adjustments
```

---

### 4️⃣ 处理基数补差
**命令：** `base:process-compensation`
**执行时间：** 每天早上 9:30
**功能：** 处理员工基数补差计算

```php
$schedule->command('base:process-compensation')
```

### 5️⃣ 生成资料交付记录
**命令：** `delivery:generate`
**执行时间：** 每月1日早上 8:00
**功能：** 自动生成本月/本季度的资料交付记录，发送新周期提醒

```php
$schedule->command('delivery:generate')
         ->monthlyOn(1, '08:00')
         ->timezone('Asia/Shanghai');
```

**等效 Cron：**
```bash
0 8 1 * * cd /your/project && php artisan delivery:generate
```

---

### 6️⃣ 检查未交付记录
**命令：** `delivery:check-pending`
**执行时间：** 每月最后一天 20:00
**功能：** 检查当月未交付的记录，发送提醒通知

```php

         });
```

**等效 Cron：**
```bash
0 20 28-31 * * [ $(date -d tomorrow +\%d) -eq 1 ] && cd /your/project && php artisan delivery:check-pending
```

## 🚀 服务器配置方法

### ✅ 方法1：统一配置（强烈推荐）⭐⭐⭐⭐⭐

**只需要配置一次，所有任务自动执行！**

```bash
# 编辑 Crontab
crontab -e

# 添加以下一行（修改项目路径）
* * * * * cd /var/www/html/re_li_zi_yuan && php artisan schedule:run >> /dev/null 2>&1
```

**优点：**
- ✅ 只需要配置一次
- ✅ 所有定时任务自动执行
- ✅ 新增定时任务无需修改 Cron
- ✅ Laravel 自动管理执行时间
- ✅ 最简单、最推荐的方式

---

### ❌ 方法2：分别配置（不推荐）

如果您坚持要分别配置每个任务：

```bash
crontab -e

0 23 28-31 * * [ $(date -d tomorrow +\%d) -eq 1 ] && cd /var/www/html/re_li_zi_yuan && php artisan insurance:generate-monthly-details >> /dev/null 2>&1

# 3. 每天早上9点处理员工调基
0 9 * * * cd /var/www/html/re_li_zi_yuan && php artisan base:process-employee-adjustments >> /dev/null 2>&1

# 4. 每天早上9点30分处理基数补差
30 9 * * * cd /var/www/html/re_li_zi_yuan && php artisan base:process-compensation >> /dev/null 2>&1

# 5. 每月1日早上8点生成资料交付记录

# 6. 每月最后一天20点检查未交付
```

**缺点：**

- ❌ 维护麻烦
