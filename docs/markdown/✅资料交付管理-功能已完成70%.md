## 📝 使用流程

### 第一步：配置项目
1. 进入"资料交付配置"页面
2. 点击"添加配置"
3. 选择项目
4. 设置交付周期（月/季度）
5. 设置交付方式（快递/电子）
6. 添加资料清单
7. 保存

### 第二步：系统自动生成记录
- 系统会在每月1日自动生成交付记录
- 发送提醒通知给经办

### 第三步：经办提交交付
**快递方式**:
1. 查看待办交付列表
2. 选择待交付记录
3. 填写快递单号和寄出日期
4. 填写资料说明
5. 提交

**电子方式**:
1. 查看待办交付列表
2. 选择待交付记录
3. 上传电子资料文件
4. 填写资料说明
5. 提交

### 第四步：系统提醒
- 每月月底检查未交付记录
- 向经办发送提醒通知
