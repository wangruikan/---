# 快速开始 - 资料交付管理

## 🚀 三步启动

### 第一步：执行SQL（1分钟）

**打开数据库管理工具，执行以下SQL文件：**
```
database/migrations/create_document_delivery_tables.sql
```

**这将创建4张表：**
- ✅ project_delivery_configs
- ✅ document_deliveries
- ✅ document_delivery_attachments
- ✅ document_delivery_reminders

### 第二步：清除缓存（10秒）

**在项目根目录执行：**
```bash
php artisan cache:clear
```

### 第三步：刷新浏览器（5秒）

**刷新前端页面，在侧边栏找到：**
```
资料交付管理
├── 交付配置
└── 交付记录
```

## 🎯 开始使用

### 1. 配置项目（第一次）

1. 点击 "交付配置"
2. 点击 "添加配置"
3. 填写表单：
   - 选择项目
   - 交付周期：按月 或 按季度
   - 交付方式：快递 或 电子
   - 添加资料清单（可选）
4. 保存

### 2. 系统自动生成记录

- 每月1日 08:00 自动生成
- 或手动执行：`php artisan delivery:generate`

### 3. 经办提交交付

1. 进入 "交付记录"
2. 找到"待交付"记录
3. 点击 "提交交付"
4. 快递：填单号+日期
   或电子：上传文件
5. 提交

## ⏰ 定时任务（可选）

**如需自动执行，配置cron任务：**
```cron
* * * * * cd /path/to/project && php artisan schedule:run >> /dev/null 2>&1
```

**或手动执行：**
```bash
# 生成交付记录
php artisan delivery:generate

# 检查未交付
php artisan delivery:check-pending
```
