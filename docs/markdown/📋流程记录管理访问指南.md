# 📋 流程记录管理访问指南

## 🎯 如何访问流程记录管理

### 📍 菜单位置
流程记录管理功能位于：
```
侧边栏 → 审批管理 → 流程记录管理
```

### 🔐 权限要求
只有以下用户可以看到和访问流程记录管理：

**✅ 可以访问的用户：**
- 超级管理员 (`super_admin`)
- 管理员 (`admin`) 
- 经理 (`manager`)
- 高级经理 (`senior_manager`)
- 审批级别2、3、4的用户

**❌ 无法访问的用户：**
- 第1个审批节点的用户
- 普通员工
- 无权限的用户

### 📊 功能说明

#### 统计概览
- **总记录数** - 所有流程记录总数
- **待处理** - 状态为"pending"的记录
- **已通过** - 状态为"approved"的记录  
- **已拒绝** - 状态为"rejected"的记录

#### 筛选功能
- **按状态筛选** - 待处理/已通过/已拒绝/已完成
- **按业务名称搜索** - 模糊搜索业务名称
- **按时间范围筛选** - 选择创建时间范围

#### 数据展示
- **业务名称** - 流程相关的业务名称
- **员工信息** - 员工姓名和身份证号
- **审批状态** - 当前审批状态
- **审批级别** - 当前处于第几级审批
- **审批人** - 负责审批的人员
- **时间信息** - 创建时间和更新时间

## 🚀 使用步骤

### 1. 登录系统
确保使用有权限的账号登录

### 2. 选择账套
在右上角选择要查看的账套

### 3. 导航到功能
点击侧边栏：审批管理 → 流程记录管理

### 4. 查看数据
- 查看统计概览
- 使用筛选条件查找特定记录
- 点击"查看详情"了解完整信息

## 🔧 故障排除

### 看不到菜单项？
1. **检查用户权限** - 确认用户角色和审批级别
2. **检查账套** - 确保已选择正确的账套
3. **刷新页面** - 尝试刷新浏览器页面

### 数据为空？
1. **检查账套数据** - 确认账套下有相关记录
2. **检查筛选条件** - 清除筛选条件查看全部数据
3. **检查权限** - 确认有查看该账套数据的权限

### 无法访问页面？
1. **检查URL** - 确保访问 `/process-records` 路径
2. **检查登录状态** - 确保已正确登录
3. **联系管理员** - 如需要权限请联系系统管理员
