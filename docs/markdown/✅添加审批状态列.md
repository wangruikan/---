# ✅ 发票申请 - 添加审批状态列

## 📋 需求

在发票申请列表中，除了显示普通的"状态"（草稿、审批中、已通过、已驳回、红冲），还需要显示"审批状态"（审批中、已通过、已驳回）。

---

## ✅ 实现方案

### 1. 前端 - 添加审批状态列

#### 文件：`src/views/InvoiceApplications/index.vue`

**添加表格列**：
```vue
<el-table-column label="审批状态" width="120" align="center">
  <template #default="{ row }">
    <el-tag 
      v-if="row.approval_status" 
      :type="getApprovalStatusType(row.approval_status)"
    >
      {{ getApprovalStatusText(row.approval_status) }}
    </el-tag>
    <span v-else class="text-muted">-</span>
  </template>
</el-table-column>
```

**添加处理方法**：
```javascript
// 获取审批状态类型
const getApprovalStatusType = (status) => {
  const typeMap = {
    pending: 'warning',    // 审批中 - 黄色
    approved: 'success',   // 已通过 - 绿色
    rejected: 'danger'     // 已驳回 - 红色
  }
  return typeMap[status] || 'info'
}

// 获取审批状态文本
const getApprovalStatusText = (status) => {
  const textMap = {
    pending: '审批中',
    approved: '已通过',
    rejected: '已驳回'
  }
  return textMap[status] || status
}
```

**添加样式**：
```css
.text-muted {
  color: #909399;
  font-size: 14px;
}
```

### 2. 后端 - 返回审批状态

#### 文件：`app/Http/Controllers/InvoiceApplicationController.php`

**加载审批实例关联**：
```php
$query = InvoiceApplication::where('account_set_id', $accountSetId)
    ->with(['submitter:id,name', 'creator:id,name', 'items', 'approvalInstance']);
```

**添加审批状态到返回数据**：
```php
// 添加总金额和审批状态
$applications->getCollection()->transform(function ($application) {
    $application->total_amount = $application->items->sum('amount');
    
    // 添加审批状态
    if ($application->approvalInstance) {
        $application->approval_status = $application->approvalInstance->status;
    } else {
        $application->approval_status = null;
    }
    
    return $application;
});
```

## 🎨 状态说明

### "状态" 列（业务状态）
- **草稿**：申请还未提交
- **审批中**：已提交，正在审批流程中
- **已通过**：审批全部通过
- **已驳回**：审批被驳回
- **红冲**：已作废（重新提交后原申请变为红冲）

### "审批状态" 列（审批流程状态）
- **审批中**（黄色）：审批流程进行中
- **已通过**（绿色）：审批流程全部通过
- **已驳回**（红色）：审批流程被驳回
- **-**（灰色）：未提交审批（草稿状态）

---

## 🔄 状态关系

| 业务状态 | 审批状态 | 说明 |
|---------|---------|------|
| 草稿 | - | 未提交审批 |
| 审批中 | 审批中 | 审批流程进行中 |
| 已通过 | 已通过 | 审批全部通过 |
| 已驳回 | 已驳回 | 审批被驳回 |
| 红冲 | - | 已作废的申请 |
