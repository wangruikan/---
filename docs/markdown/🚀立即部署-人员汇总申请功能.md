## 📋 功能说明

### 核心改变：

#### ❌ 之前的逻辑：
```
导入Excel → 部分导入成功 → 记录差异
```

#### ✅ 现在的逻辑：
```
导入Excel → 检测人员差异 
  ├─ 无差异 → 导入成功
  └─ 有差异 → 拒绝导入 + 创建变动记录
```

### 业务流程：

```
1️⃣ 导入Excel
   ↓
2️⃣ 发现人员不匹配
   ↓
3️⃣ ❌ 拒绝导入
   ↓
4️⃣ 📝 自动创建人员变动记录
   ↓
5️⃣ 💬 提示："请先维护人员档案"
   ↓
6️⃣ 到"人员汇总申请"查看
   ↓
7️⃣ 上传附件、提交审批（跳过第一节点）
   ↓
8️⃣ 审批通过
   ↓
9️⃣ 维护员工档案（新增/删除）
   ↓
🔟 重新生成工资表
   ↓
1️⃣1️⃣ 重新导入Excel → ✅ 成功
```

### 问题2：Excel导入报错"未找到身份证列"
**检查：**
- Excel第2行是否包含"身份证号码"列
- 列名是否准确（支持：身份证、身份证号、身份证号码）

### 问题3：提示"Excel中的身份证号在系统中均未找到"
**解决：**
- 检查员工管理中是否有这些员工
- 检查员工的身份证号是否与Excel一致
- 先添加员工，再导入

### 问题4：人员汇总申请菜单看不到
**解决：**
```
1. 清理浏览器缓存（Ctrl + F5）
2. 检查是否已分配账套
3. 检查用户角色权限
```
