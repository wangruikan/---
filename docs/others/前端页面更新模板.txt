# 前端页面更新模板

## 需要更新的页面（共5个）

每个页面需要做3个修改：

### 修改 1: 模板开头添加提示组件
```vue
<template>
  <div class="xxx-page">
    <NoAccountSetTip v-if="!isAdmin && !currentAccountSetId" />
    <div v-else>
    <!-- 原有内容 -->
```

### 修改 2: 模板结尾闭合标签
```vue
    </div>  <!-- 闭合 v-else -->
  </div>
</template>
```

### 修改 3: script 部分添加导入和变量
```vue
<script setup>
import { ..., computed } from 'vue'  // 添加 computed
import { useUserStore } from '@/stores/user'
import { useAccountSetStore } from '@/stores/accountSet'
import NoAccountSetTip from '@/components/NoAccountSetTip.vue'

const userStore = useUserStore()
const accountSetStore = useAccountSetStore()

const isAdmin = computed(() => userStore.userInfo?.role === 'admin')
const currentAccountSetId = computed(() => accountSetStore.currentAccountSetId)
```
