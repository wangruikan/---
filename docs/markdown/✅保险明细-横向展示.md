# ✅ 保险明细 - 横向展示

## 📋 需求

保险明细不要用展开行（expand），而是直接在工资表中横向显示，与其他列保持在同一行。

---

## 🔧 修改内容

### 1. **删除展开行（expand）** ✅

移除了 `<el-table-column type="expand">` 及其内部的所有保险明细表格。

### 2. **添加横向保险明细列** ✅

在"应发工资"之后，添加了以下列组：

#### 社保明细
```
社保明细
├── 险种 (100px)
├── 基数 (90px)
├── 单位
│   ├── 比例 (70px)
│   └── 金额 (90px)
└── 个人
    ├── 比例 (70px)
    └── 金额 (90px)
```

**特点**：
- 支持多个险种（养老、医疗、失业等）
- 每个险种占一行，使用 `v-for` 循环显示
- `line-height: 24px` 保证行高一致

#### 公积金明细
```
公积金
├── 基数 (90px)
├── 单位
│   ├── 比例 (70px)
│   └── 金额 (90px)
└── 个人
    ├── 比例 (70px)
    └── 金额 (90px)
```

#### 大额医疗
```
大额医疗
├── 单位 (90px)
└── 个人 (90px)
```

#### 保险合计
```
- 单位保险合计 (130px)
- 个人保险合计 (130px)
```

## 💻 核心代码

### 社保明细（支持多险种）

```vue
<el-table-column label="险种" width="100" align="center">
  <template #default="{ row }">
    <div v-if="row.insurance_details && row.insurance_details.social_security">
      <div v-for="(item, index) in row.insurance_details.social_security" :key="index" style="line-height: 24px;">
        {{ item.name }}
      </div>
    </div>
    <span v-else>-</span>
  </template>
</el-table-column>
```

### 公积金明细（单项）

```vue
<el-table-column label="基数" width="90" align="right">
  <template #default="{ row }">
    {{ row.insurance_details && row.insurance_details.housing_fund ? formatMoney(row.insurance_details.housing_fund.base) : '-' }}
  </template>
</el-table-column>
```

## ✅ 优势

### 修改前（展开行）
- ❌ 需要点击展开按钮才能看到明细
- ❌ 占用额外的垂直空间
- ❌ 不便于快速对比多个员工的保险数据

### 修改后（横向展示）
- ✅ 所有数据在同一行，一目了然
- ✅ 便于横向对比不同员工的保险情况
- ✅ 符合传统工资表的展示习惯
- ✅ 可以直接导出为 Excel，格式更标准
