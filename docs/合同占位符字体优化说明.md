# 合同占位符字体优化说明

## 优化内容

### 问题描述
合同管理中通过占位符自动填充的数据：
- ❌ 字体太小，不够清晰
- ❌ 字体加粗（bold），与正文不一致
- ❌ 有阴影效果，显得模糊

### 优化方案
调整自动填充数据的字体格式，使其与正文保持一致：
- ✅ 字体更大（最小 14px，最大 20px）
- ✅ 字体正常（不加粗）
- ✅ 无阴影效果，更清晰

## 修改详情

### 文件位置
`src/utils/pdfFillService.js`

### 修改内容

#### 1. 字体大小调整

**修改前：**
```javascript
fontSize: Math.max(Math.min(position.height * 0.8, 18), 12)
// 最小 12px，最大 18px
```

**修改后：**
```javascript
fontSize: Math.max(Math.min(position.height * 0.9, 20), 14)
// 最小 14px，最大 20px，整体更大
```

#### 2. 字体样式调整

**修改前：**
```javascript
ctx.font = `bold ${actualFontSize}px ${fontFamily}, "Microsoft YaHei", "SimSun", sans-serif`
// 使用粗体（bold）
```

**修改后：**
```javascript
ctx.font = `${actualFontSize}px ${fontFamily}, "Microsoft YaHei", "SimSun", sans-serif`
// 使用正常字体（不加粗）
```

#### 3. 移除阴影效果

**修改前：**
```javascript
// 绘制文字（添加轻微阴影提高清晰度）
ctx.shadowColor = 'rgba(0, 0, 0, 0.1)'
ctx.shadowBlur = 0.5
ctx.shadowOffsetX = 0.5
ctx.shadowOffsetY = 0.5
ctx.fillText(text, textX, textY)

// 清除阴影
ctx.shadowColor = 'transparent'
ctx.shadowBlur = 0
ctx.shadowOffsetX = 0
ctx.shadowOffsetY = 0
```

**修改后：**
```javascript
// 绘制文字（清晰无阴影）
ctx.fillText(text, textX, textY)
```

- ✅ 手机号

- ✅ 其他所有占位符字段

## 技术说明

### 字体渲染优化

1. **高分辨率渲染**
   - 使用 2倍分辨率 Canvas
   - 确保文字清晰度

2. **抗锯齿处理**
   ```javascript
   ctx.imageSmoothingEnabled = true
   ctx.imageSmoothingQuality = 'high'
   ```

3. **字体回退**
   ```javascript
   "Arial", "Microsoft YaHei", "SimSun", sans-serif
   ```
   - 优先使用 Arial
   - 中文使用微软雅黑或宋体
   - 确保跨平台兼容

### 字体大小计算

```javascript
fontSize = Math.max(Math.min(position.height * 0.9, 20), 14)
```

- `position.height * 0.9`：根据占位符高度自适应
- `Math.min(..., 20)`：最大不超过 20px
- `Math.max(..., 14)`：最小不低于 14px

## 使用说明

### 自动生效
修改后，所有新创建或重新填充的合同都会使用新的字体格式。

### 测试步骤

1. **创建新合同**
   - 进入"合同管理"
   - 选择合同模板
   - 填写员工信息
   - 点击"使用占位符自动填充"

2. **查看效果**
   - 打开生成的合同PDF
   - 检查自动填充的字段
   - 对比正文字体

3. **验证清晰度**
   - 放大PDF查看
   - 确认文字清晰
   - 确认字体大小合适

## 注意事项

1. **已有合同**
   - 已生成的合同不会自动更新
   - 需要重新填充才能应用新格式

2. **占位符位置**
   - 字体变大后，确保占位符区域足够大
   - 如果文字超出，需要调整占位符位置

3. **字体兼容性**
   - 使用标准字体确保兼容性
   - 支持中英文混合

4. **PDF质量**
   - 使用高分辨率渲染
   - 导出质量设为最高（1.0）

## 常见问题

### Q1: 字体还是太小怎么办？
A: 可以进一步调整字体大小参数：
```javascript
fontSize: Math.max(Math.min(position.height * 0.95, 22), 16)
// 最小 16px，最大 22px
```

### Q2: 字体太大超出占位符区域？
A: 需要调整占位符位置，增大占位符区域的宽度和高度。

### Q3: 中文显示不正常？
A: 确保字体回退列表包含中文字体：
```javascript
"Microsoft YaHei", "SimSun", "SimHei"
```

### Q4: 如何让字体更粗？
A: 如果需要加粗，可以改回：
```javascript
ctx.font = `bold ${actualFontSize}px ...`
```
