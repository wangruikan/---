## 📊 公积金补差与社保补差的差异

| 对比项 | 社保补差 | 公积金补差 |
|--------|---------|------------|
| 触发时机 | 修改社保地区上下限 | 修改公积金配置上下限 |
| 补差类型 | `social_security` | `housing_fund` |
| 细分类型 | 有（养老、医疗等） | 无（只有一个配置） |
| 表格列 | 动态列（每个险种） | 固定列（单位、个人） |
| compensation_details | 数组，多个类型 | 数组，只有1个元素 |

## 🎯 使用方式

### 修改公积金上下限触发补差

1. 进入"公积金管理"页面
2. 选择某个地区的配置
3. 修改上下限（例如：2000-25000 → 3000-30000）
4. 保存后自动触发补差计算
5. 在"参保增减管理" → "明细" → "公积金补交明细"查看

### 查看公积金补差明细

1. 进入"保险管理" → "参保增减管理"
2. 点击"明细"主标签
3. 点击"公积金补交明细"子标签
4. 查看所有公积金补差记录

## 📝 数据格式

