# 累计专项扣除（社保公积金）- 功能说明

## 📝 功能概述

**累计专项扣除** = 当年1月到本月的（个人社保 + 个人公积金）累计值

这个字段用于个人所得税计算，是社保公积金个人部分的累计扣除额。

---

## 🎯 计算规则

### 基本公式
```
累计专项扣除 = ∑(个人社保) + ∑(个人公积金)
```

**查询范围**：当年1月到本月（含本月）

### 计算步骤

#### 1️⃣ 查询之前月份的数据
```php
$previousSalaries = Salary::where('employee_id', $employee->id)
    ->where('account_set_id', $accountSetId)
    ->where('month', '>=', $year . '-01')  // 当年1月开始
    ->where('month', '<', $month)           // 到本月之前
    ->get();
```

#### 2️⃣ 累加之前月份的社保和公积金
```php
// 之前月份累计
$previousSocialSecurity = $previousSalaries->sum('social_security');
$previousHousingFund = $previousSalaries->sum('housing_fund');

// 加上本月的
$cumulativeSocialSecurity = $previousSocialSecurity + $currentMonthSocialSecurity;
$cumulativeHousingFund = $previousHousingFund + $currentMonthHousingFund;
```

#### 3️⃣ 计算累计专项扣除
```php
$cumulativeSpecialDeductionInsurance = $cumulativeSocialSecurity + $cumulativeHousingFund;
```

## 💡 具体案例

### 案例1：正常缴纳社保公积金的员工

假设某员工2025年每月社保公积金如下：

| 月份 | 个人社保 | 个人公积金 | 当月合计 |
|-----|---------|-----------|---------|
| 1月 | ¥800 | ¥600 | ¥1,400 |
| 2月 | ¥800 | ¥600 | ¥1,400 |
| 3月 | ¥800 | ¥600 | ¥1,400 |
| 4月 | ¥800 | ¥600 | ¥1,400 |
| 5月 | ¥800 | ¥600 | ¥1,400 |
| 6月 | ¥800 | ¥600 | ¥1,400 |
| 7月 | ¥800 | ¥600 | ¥1,400 |
| 8月 | ¥800 | ¥600 | ¥1,400 |
| 9月 | ¥800 | ¥600 | ¥1,400 |

**计算2025年9月工资时**：
```
累计社保 = 800 × 9 = 7,200元
累计公积金 = 600 × 9 = 5,400元
累计专项扣除 = 7,200 + 5,400 = 12,600元
```

### 案例2：年中调整基数的员工

| 月份 | 个人社保 | 个人公积金 | 说明 |
|-----|---------|-----------|------|
| 1-6月 | ¥800 | ¥600 | 旧基数 |
| 7月 | ¥900 | ¥700 | 调整基数 |
| 8月 | ¥900 | ¥700 | 新基数 |
| 9月 | ¥900 | ¥700 | 新基数 |

**计算2025年9月工资时**：
```
累计社保 = (800 × 6) + (900 × 3) = 4,800 + 2,700 = 7,500元
累计公积金 = (600 × 6) + (700 × 3) = 3,600 + 2,100 = 5,700元
累计专项扣除 = 7,500 + 5,700 = 13,200元
```

### 案例3：5月入职的新员工

假设员工2025年5月入职，系统只从5月开始有工资记录：

| 月份 | 个人社保 | 个人公积金 |
|-----|---------|-----------|
| 5月 | ¥800 | ¥600 |
| 6月 | ¥800 | ¥600 |
| 7月 | ¥800 | ¥600 |
| 8月 | ¥800 | ¥600 |
| 9月 | ¥800 | ¥600 |

**计算2025年9月工资时**：
```
累计社保 = 800 × 5 = 4,000元
累计公积金 = 600 × 5 = 3,000元
累计专项扣除 = 4,000 + 3,000 = 7,000元
```

## 🔧 实现细节

### 数据库字段
```sql
ALTER TABLE `salaries` 
ADD COLUMN `cumulative_special_deduction_insurance` DECIMAL(10, 2) DEFAULT 0 
COMMENT '累计专项扣除（社保公积金个人部分）' 
AFTER `cumulative_basic_deduction`;
```

### 后端逻辑
**文件**: `app/Http/Controllers/SalaryController.php`

```php
// 1. 查询当年之前月份的工资记录
$previousSalaries = Salary::where('employee_id', $employee->id)
    ->where('account_set_id', $accountSetId)
    ->where('month', '>=', $year . '-01')
    ->where('month', '<', $month)
    ->get();

// 2. 累计社保公积金 = 之前月份 + 当前月份
$cumulativeSocialSecurity = $previousSalaries->sum('social_security') 
    + $insuranceData['personal_social_security'];
$cumulativeHousingFund = $previousSalaries->sum('housing_fund') 
    + $insuranceData['personal_housing_fund'];

// 3. 累计专项扣除（社保公积金个人部分）= 累计社保 + 累计公积金
$cumulativeSpecialDeductionInsurance = $cumulativeSocialSecurity + $cumulativeHousingFund;

// 4. 保存到数据库
$salary = Salary::create([
    // ... 其他字段
    'cumulative_special_deduction_insurance' => $cumulativeSpecialDeductionInsurance,
    // ... 其他字段
]);
```

## 📊 在个税计算中的应用

累计专项扣除是计算累计应纳税所得额的重要组成部分：

```
累计应纳税所得额 = 累计收入 
                - 累计减除费用（5000×月数）
                - 累计专项扣除（社保公积金）← 新增！
                - 累计专项附加扣除
```

**示例**：
```
累计收入：100,000元
累计减除费用：45,000元（5000×9）
累计专项扣除：12,600元（社保7200 + 公积金5400）← 新增！
累计专项附加扣除：5,000元

累计应纳税所得额 = 100,000 - 45,000 - 12,600 - 5,000 = 37,400元
```

## 🔄 与其他累计字段的关系

在工资表中，现在有以下累计字段：

| 字段名 | 说明 | 计算方式 |
|-------|------|---------|
| **累计收入** | 当年累计应发工资 | ∑(1月到本月的应发工资) |
| **累计减除费用** | 当年累计基本减除费用 | 5000 × 实际工作月数 |
| **累计专项扣除** ← 新增 | 当年累计社保公积金 | ∑(个人社保) + ∑(个人公积金) |
| **累计专项附加扣除** | 当年累计专项附加扣除 | ∑(1月到本月的6项扣除) |

这些字段共同用于计算累计应纳税所得额。
