# ✅ 完成 - 重新发起不创建新记录

## 📋 需求

红冲状态的申请重新发起时，不需要创建新的申请记录，直接在原记录上重新发起审批流程即可。

---

## ✅ 实现方案

### 核心逻辑

**简化流程**：
1. 红冲状态 → 显示"重新发起"按钮，可以编辑
2. 点击"重新发起" → 在原记录上创建新的审批实例，状态改为"审批中"
3. 审批中 → "重新发起"按钮自然隐藏（因为不是红冲状态了）
4. 如果再次被驳回 → 状态又变为"红冲"，又可以编辑和重新发起

**不需要额外字段**：
- 不需要 `has_resubmitted` 标记
- 不需要 `new_application_id` 关联
- **状态本身就足够判断**！

### 1. 控制器 - 简化重新发起逻辑

#### 文件：`app/Http/Controllers/InvoiceApplicationController.php`

**修改前**（创建新记录）：
```php
public function resubmit($id)
{
    // ...
    
    // 创建新的申请（复制数据）
    $newApplication = InvoiceApplication::create([
        'account_set_id' => $application->account_set_id,
        'application_no' => InvoiceApplication::generateApplicationNo(),
        // ... 复制所有字段
    ]);
    
    // 复制明细项
    foreach ($application->items as $item) {
        InvoiceItem::create([
            'application_id' => $newApplication->id,
            // ...
        ]);
    }
    
    // 创建审批实例，关联新申请
    $instance = ApprovalInstance::create([
        'business_id' => $newApplication->id,
        // ...
    ]);
}
```

**修改后**（直接在原记录上发起）：
```php
public function resubmit($id)
{
    $application = InvoiceApplication::with('items')->find($id);
    
    // 验证...
    
    DB::beginTransaction();
    try {
        // 获取审批人配置
        $approvers = DB::table('account_set_users')
            ->where('account_set_id', $application->account_set_id)
            ->where('approval_level', '>', 1)
            ->orderBy('approval_level')
            ->get();
        
        // 创建新的审批实例（关联原申请）
        $instance = ApprovalInstance::create([
            'account_set_id' => $application->account_set_id,
            'business_type' => '发票申请（重新提交）',
            'business_id' => $application->id,  // ← 关联原申请ID
            'current_step' => 2,
            'total_steps' => $approvers->count() + 1,
            'status' => 'pending',
            'created_by' => Auth::id(),
        ]);
        
        // 添加附件...
        // 创建审批记录...
        
        // 更新原申请状态为"审批中"
        $application->update([
            'status' => InvoiceApplication::STATUS_PENDING,
            'approval_instance_id' => $instance->id,
            'submitted_at' => now(),
        ]);
        
        DB::commit();
        
        return response()->json([
            'success' => true,
            'message' => '重新发起成功，已提交审批',
            'data' => $application->load(['items', 'approvalInstance'])
        ]);
    }
}
```

**关键变化**：
- ✅ 不创建新申请记录
- ✅ 不复制明细项（原明细项保持不变）
- ✅ 创建新审批实例，关联原申请
- ✅ 更新原申请状态为"审批中"

```php
/**
 * 是否可以编辑
 */
public function canEdit()
{
    // 草稿、正常、红冲状态都可以编辑
    return in_array($this->status, [
        self::STATUS_DRAFT, 
        self::STATUS_NORMAL, 
        self::STATUS_RED_FLUSHED
    ]);
}

/**
 * 是否可以重新提交（重新发起）
 */
public function canResubmit()
{
    // 只有红冲状态可以重新发起
    return $this->status === self::STATUS_RED_FLUSHED;
}
```

```php
case '发票申请（重新提交）':
    $invoiceApplication = \App\Models\InvoiceApplication::find($businessId);
    if ($invoiceApplication) {
        if ($status === 'completed') {
            $invoiceApplication->update([
                'status' => 'approved',
            ]);
        } elseif ($status === 'rejected') {
            // 驳回时改为红冲状态
            $invoiceApplication->update([
                'status' => 'red_flushed',
            ]);
        } elseif ($status === 'in_approval') {
            $invoiceApplication->update([
                'status' => 'pending',
            ]);
        }
    }
    break;
```
