## 🎯 使用流程

### 用户操作步骤：

1. **发起付款**
   - 点击"发起付款"按钮
   - 填写付款信息
   - 上传PDF附件或生成PDF表格

2. **签名盖章**
   - 点击"签名盖章"按钮
   - 从列表中选择要签名的PDF文件
   - 点击"确认并打开签名编辑器"

3. **编辑签名**
   - 在右侧选择签名或印章
   - 点击PDF文档上的位置添加签名/印章
   - 可以拖动、调整大小、删除签名/印章
   - 点击"保存签名后的PDF"

4. **完成**
   - 签名后的PDF自动替换原文件
   - 文件名会添加"_已签名_时间戳"后缀
   - 继续提交付款申请
