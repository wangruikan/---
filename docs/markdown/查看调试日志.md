# 查看调试日志

请按以下步骤操作：

## 1. 重新确认处理

1. 打开浏览器，进入 "保险管理" → "保险增减"
2. 找到 wrk 的待处理记录
3. 点击 "确认处理"

## 2. 查看调试日志

在 PowerShell 中执行：

```powershell
Get-Content storage/logs/base_constraint_debug.log
```

这个日志会显示：
- 约束逻辑是否被执行
- 原始基数是多少
- 约束后的基数是多少

## 3. 查看数据库

```sql
SELECT 
    employee_name,
    employee_social_security_base,
    employee_medical_insurance_base,
    employee_housing_fund_base,
    last_updated_at
FROM insurance_personnel
WHERE employee_name = 'wrk';
```

如果调试日志中没有内容，说明约束方法没有被调用，我们需要检查其他地方。
