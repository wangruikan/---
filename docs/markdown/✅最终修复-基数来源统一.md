## 🔑 核心原则

> **基数从表字段读取，比例从 JSON 读取**

这样确保：
1. ✅ 基数一致性 - 所有接口使用相同的基数
2. ✅ 比例灵活性 - JSON 可以存储复杂的比例配置
3. ✅ 数据可靠性 - 表字段是单一数据源
