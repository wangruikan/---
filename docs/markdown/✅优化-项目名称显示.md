# ✅ 优化：项目名称超长显示

## 🎯 问题描述

当选择多个项目生成工资表时（如"公园项目、老年公寓、物业管理"），项目名称会很长，导致界面显示不美观。

## 💡 解决方案

使用 **文本截断 + 悬停提示（Tooltip）** 的方式优化显示：

1. **表格中的项目名称**：超过 20 个字符时截断，显示省略号
2. **对话框标题**：超过 30 个字符时截断，显示省略号
3. **统计信息**：超过 40 个字符时截断，显示省略号
4. **悬停显示**：鼠标悬停在截断的文字上时，显示完整的项目名称

## 📝 修改内容

### 1. 表格列 - 项目名称

```vue
<el-table-column prop="project_name" label="项目名称" min-width="200" show-overflow-tooltip>
  <template #default="scope">
    <el-tooltip 
      :content="scope.row.project_name" 
      placement="top" 
      :disabled="scope.row.project_name.length <= 20"
    >
      <span>
        {{ scope.row.project_name.length > 20 
           ? scope.row.project_name.substring(0, 20) + '...' 
           : scope.row.project_name }}
      </span>
    </el-tooltip>
  </template>
</el-table-column>
```

**效果**：
- 短名称：`公园项目` → 直接显示，无提示
- 长名称：`公园项目、老年公寓、物业管理` → 显示 `公园项目、老年公寓、物业管...`
- 悬停时：显示完整名称

### 2. 对话框标题

```vue
<template #header>
  <div class="dialog-header-title">
    <span class="title-text">工资明细 - </span>
    <el-tooltip 
      :content="currentSheet?.project_name" 
      placement="top"
      :disabled="!currentSheet?.project_name || currentSheet.project_name.length <= 30"
    >
      <span class="project-name">
        {{ currentSheet?.project_name && currentSheet.project_name.length > 30 
           ? currentSheet.project_name.substring(0, 30) + '...' 
           : currentSheet?.project_name }}
      </span>
    </el-tooltip>
    <span class="month-text"> ({{ currentSheet?.month }})</span>
  </div>
</template>
```

**效果**：
```
工资明细 - 公园项目、老年公寓、物业管理、社区... (2025-11)
           ↑ 悬停显示完整名称
```

### 3. 统计信息

```vue
<el-descriptions-item label="项目名称">
  <el-tooltip 
    :content="currentSheet.project_name" 
    placement="top"
    :disabled="currentSheet.project_name.length <= 40"
  >
    <span>
      {{ currentSheet.project_name.length > 40 
         ? currentSheet.project_name.substring(0, 40) + '...' 
         : currentSheet.project_name }}
    </span>
  </el-tooltip>
</el-descriptions-item>
```

```css

```

## 📏 截断规则

| 位置 | 最大长度 | 超出显示 | 原因 |
|------|---------|---------|------|
| 表格列 | 20字符 | `...` | 表格空间有限 |
| 对话框标题 | 30字符 | `...` | 标题栏宽度限制 |
| 统计信息 | 40字符 | `...` | 描述列表空间较大 |

## ✨ 用户体验提升

1. **界面整洁**：长名称不会撑坏布局
2. **信息不丢失**：悬停即可看到完整内容
3. **视觉引导**：`cursor: help` 光标提示用户可以悬停
4. **颜色区分**：标题中的项目名称用蓝色突出显示
5. **智能判断**：只有超长才截断，短名称不受影响

## 🔄 兼容性

- ✅ 单项目：完全正常显示（不截断）
- ✅ 多项目（短）：如"项目A、项目B" → 正常显示
- ✅ 多项目（长）：自动截断，悬停显示完整
