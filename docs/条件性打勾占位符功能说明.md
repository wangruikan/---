# 条件性打勾占位符功能说明

## 功能概述

系统支持在合同模板中设置条件性打勾占位符，根据员工的实际信息自动在相应位置显示 ✓ 符号。

## 可用的条件性打勾字段

### 1. 性别打勾字段

| 占位符字段 | 显示名称 | 触发条件 | 显示内容 |
|-----------|---------|---------|---------|
| `gender_male_check` | 性别-男（打勾） | 员工性别为"男" | ✓ |
| `gender_female_check` | 性别-女（打勾） | 员工性别为"女" | ✓ |

**逻辑规则：**
- 如果员工性别是"男"，则 `gender_male_check` 显示 ✓，`gender_female_check` 为空
- 如果员工性别是"女"，则 `gender_female_check` 显示 ✓，`gender_male_check` 为空

### 2. 户口类型打勾字段

| 占位符字段 | 显示名称 | 触发条件 | 显示内容 |
|-----------|---------|---------|---------|
| `household_agricultural_check` | 户籍-农业（打勾） | 户口类型为"农业" | ✓ |
| `household_non_agricultural_check` | 户籍-非农业（打勾） | 户口类型为"非农业" | ✓ |

**逻辑规则：**
- 如果户口类型是"农业"，则 `household_agricultural_check` 显示 ✓，`household_non_agricultural_check` 为空
- 如果户口类型是"非农业"，则 `household_non_agricultural_check` 显示 ✓，`household_agricultural_check` 为空

## 使用方法

### 1. 在合同模板中设置占位符

1. 进入"项目管理"
2. 点击项目的"占位符字段"按钮
3. 勾选需要的打勾字段：
   - 性别-男（打勾）
   - 性别-女（打勾）
   - 户籍-农业（打勾）
   - 户籍-非农业（打勾）
4. 保存配置

### 2. 在PDF模板中设置占位符位置

1. 进入"项目管理"
2. 点击项目的"须知设置"或"资料配置"
3. 选择合同模板
4. 点击"设置占位符"
5. 在PDF中标记4个位置：
   - 在"□男"的方框位置设置 `gender_male_check`
   - 在"□女"的方框位置设置 `gender_female_check`
   - 在"□农业"的方框位置设置 `household_agricultural_check`
   - 在"□非农业"的方框位置设置 `household_non_agricultural_check`

### 3. 生成合同

当为员工生成合同时，系统会自动：
1. 读取员工的性别和户口类型
2. 根据实际值在相应位置填充 ✓
3. 其他位置保持空白

## 示例

### 示例1：男性、非农业户口

**员工信息：**
- 性别：男
- 户口类型：非农业

**合同显示：**
```
性别：☑男  □女    户籍性质：□农业  ☑非农业
```

### 示例2：女性、农业户口

**员工信息：**
- 性别：女
- 户口类型：农业

**合同显示：**
```
性别：□男  ☑女    户籍性质：☑农业  □非农业
```

## 技术实现

### 后端逻辑

在 `EmployeeContractController.php` 中，生成合同时会根据员工信息计算这些字段的值：

```php
'gender_male_check' => $employee->gender === 'male' ? '√' : '',
'gender_female_check' => $employee->gender === 'female' ? '√' : '',
'household_agricultural_check' => $employee->household_type === 'agricultural' ? '√' : '',
'household_non_agricultural_check' => $employee->household_type === 'non_agricultural' ? '√' : '',
```

### 前端配置

在项目管理的占位符字段配置中，这些字段会显示为：
- 性别-男（打勾）
- 性别-女（打勾）
- 户籍-农业（打勾）
- 户籍-非农业（打勾）

## 注意事项

1. **必须设置所有相关字段**：如果要使用性别打勾功能，建议同时设置"男"和"女"两个占位符
2. **位置要准确**：在PDF中设置占位符位置时，要确保位置对应到正确的方框
3. **员工信息要完整**：确保员工档案中的性别和户口类型字段已填写
4. **字体支持**：确保PDF使用的字体支持 ✓ 符号的显示
