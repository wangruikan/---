# ✅ 完成 - 工资表备注事项功能

## 📋 需求

在薪资计算模块中添加备注事项功能：

1. **备注事项管理模块**（二级菜单）
   - 显示所有项目列表
   - 可以为每个项目按月份设置工资表备注
   - 只有审批节点2、3、4的人可以访问和管理

2. **工资表列表增强**
   - 添加"备注事项"列
   - 点击查看备注，根据工资表的项目和月份显示对应的备注
   - 多个项目时，弹窗显示备注列表

---

## ✅ 实现方案

### 数据结构

```
工资表备注表 (payroll_remarks)
├─ id - 主键
├─ account_set_id - 账套ID
├─ project_name - 项目名称
├─ year - 年份
├─ month - 月份
├─ remark - 工资表备注内容
├─ created_by - 创建人
├─ updated_by - 更新人
└─ timestamps - 时间戳

唯一索引：(account_set_id, project_name, year, month)
```

### 核心逻辑

**备注管理**：
- 按账套、项目、年月唯一标识一条备注
- 创建或更新时自动判断（upsert逻辑）
- 权限控制：只有审批级别2、3、4可以访问

**工资表显示**：
- 解析工资表的项目名称（可能多个，用"、"分隔）
- 根据年月和项目名称查询对应的备注
- 弹窗展示所有相关项目的备注

Schema::create('payroll_remarks', function (Blueprint $table) {
    $table->id();
    $table->unsignedBigInteger('account_set_id')->comment('账套ID');
    $table->string('project_name', 100)->comment('项目名称');
    $table->integer('year')->comment('年份');
    $table->integer('month')->comment('月份');
    $table->text('remark')->nullable()->comment('工资表备注');
    $table->unsignedBigInteger('created_by')->nullable()->comment('创建人');
    $table->unsignedBigInteger('updated_by')->nullable()->comment('更新人');
    $table->timestamps();
    
    // 索引
    $table->index(['account_set_id', 'project_name', 'year', 'month'], 'idx_payroll_remark');
    $table->unique(['account_set_id', 'project_name', 'year', 'month'], 'unique_payroll_remark');
});

public function getByProjectAndPeriod(Request $request)
{
    $projectNames = $request->input('project_names', []); // 项目名称数组
    $year = $request->input('year');
    $month = $request->input('month');
    
    $remarks = PayrollRemark::where('account_set_id', $accountSetId)

        'success' => true,

    ]);
}

const handleViewRemarks = async (row) => {
  try {
    // 解析工资期间（格式：2025-11）
    const [year, month] = row.month.split('-')
    // 解析项目名称（可能有多个，用、分隔）
    const projectNames = row.project_name.split('、')
    
    const response = await request({
      url: '/payroll-remarks/get-by-project-period',
      method: 'post',
      data: {
        account_set_id: accountSetStore.currentAccountSetId,
        project_names: projectNames,
        year: parseInt(year),
        month: parseInt(month)
      }
    })
    
    if (response.success) {
      currentRemarks.value = response.data || []
      remarksDialogVisible.value = true
    }
  } catch (error) {
    console.error('加载备注失败', error)
    ElMessage.error('加载备注失败')
  }
}

## 📊 使用流程

### 流程1：设置工资表备注

1. **进入备注事项管理**
   - 侧边栏：薪金计算 → 备注事项
   - 权限：只有审批节点2、3、4可以访问

2. **添加备注**
   - 点击"添加备注"按钮
   - 选择项目名称（从已有项目中选择）
   - 选择年份和月份
   - 输入工资表备注内容（最多500字）
   - 点击"确定"保存

3. **编辑备注**
   - 在列表中找到要编辑的备注
   - 点击"编辑"按钮
   - 修改备注内容（项目、年月不可修改）
   - 点击"确定"保存

4. **删除备注**
   - 点击"删除"按钮
   - 确认删除

### 流程2：查看工资表备注

1. **进入工资管理**
   - 侧边栏：薪金计算 → 薪金计算

2. **查看备注**
   - 在工资表列表中找到要查看的工资表
   - 点击"备注事项"列的"查看备注"按钮
   - 弹窗显示该工资表所有项目的备注信息

3. **备注显示规则**
   - 单个项目：显示该项目对应月份的备注
   - 多个项目：显示所有相关项目的备注列表
   - 无备注：显示"暂无备注信息"
