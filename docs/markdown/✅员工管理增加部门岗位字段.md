## 🎯 功能说明

### 部门字段
- **用途**：员工所属部门
- **示例**：技术部、财务部、行政部
- **可选**：是（可以为空）
- **录入位置**：员工档案编辑页面

### 岗位字段
- **用途**：员工具体岗位
- **示例**：Java工程师、会计、行政助理
- **可选**：是（可以为空）
- **录入位置**：员工档案编辑页面

---

## 🔄 与工资表的关联

### 数据流向

```
员工管理页面
  ↓ 填写部门和岗位
  
employees 表
  ↓ 保存到数据库
  
生成工资表时
  ↓ 自动读取
  
salaries 表
  ↓ 复制到工资记录
  
工资明细显示
  ↓ 展示给用户
  
序号 | 姓名 | 部门 | 岗位 | 基本工资 | ...
 1   | 张三 | 技术 | Java |  10000  | ...
```
