## 🗄️ 现有数据库结构

### `salaries` 表（已存在）

```sql
CREATE TABLE `salaries` (
  `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
  `employee_id` BIGINT UNSIGNED,           -- 员工ID
  `project_id` BIGINT UNSIGNED,            -- 项目ID
  `month` VARCHAR(255),                    -- 工资期间（格式：2025-10）
  
  -- 工资构成
  `basic_salary` DECIMAL(10,2) DEFAULT 0,  -- 基本工资
  `allowance` DECIMAL(10,2) DEFAULT 0,     -- 津贴
  `overtime_pay` DECIMAL(10,2) DEFAULT 0,  -- 加班费
  `bonus` DECIMAL(10,2) DEFAULT 0,         -- 奖金
  `gross_salary` DECIMAL(10,2) DEFAULT 0,  -- 应发工资
  
  -- 扣款
  `social_security` DECIMAL(10,2) DEFAULT 0,    -- 社保个人部分
  `housing_fund` DECIMAL(10,2) DEFAULT 0,       -- 公积金个人部分
  `special_deduction` DECIMAL(10,2) DEFAULT 0,  -- 专项扣除
  
  -- 个税
  `taxable_income` DECIMAL(10,2) DEFAULT 0,  -- 应纳税所得额
  `personal_tax` DECIMAL(10,2) DEFAULT 0,    -- 应缴个税
  `actual_tax` DECIMAL(10,2) DEFAULT 0,      -- 实际个税
  
  -- 实发
  `net_salary` DECIMAL(10,2) DEFAULT 0,   -- 实发工资
  `paid_salary` DECIMAL(10,2) DEFAULT 0,  -- 实际发放
  
  -- 状态管理
  `status` ENUM('draft', 'submitted', 'approved', 'paid', 'rejected'),
  `submitted_by` BIGINT UNSIGNED NULL,
  `approved_by` BIGINT UNSIGNED NULL,
  `submitted_at` TIMESTAMP NULL,
  `approved_at` TIMESTAMP NULL,
  `paid_at` TIMESTAMP NULL,
  `rejection_reason` TEXT NULL,
  
  `created_at` TIMESTAMP,
  `updated_at` TIMESTAMP,
  
  FOREIGN KEY (`employee_id`) REFERENCES `employees`(`id`) ON DELETE CASCADE,
  FOREIGN KEY (`project_id`) REFERENCES `projects`(`id`) ON DELETE CASCADE
);
```

### 特点
- ✅ **一条记录 = 一个员工的一个月工资**
- ✅ 字段齐全，支持完整的工资计算
- ✅ 状态流转完整：草稿 → 提交 → 审批 → 发放

### 可选执行
```sql
-- 创建统计视图
CREATE OR REPLACE VIEW `salary_summary_by_project` AS
SELECT 
    s.project_id,
    p.name AS project_name,
    s.month,
    COUNT(DISTINCT s.employee_id) AS employee_count,
    SUM(s.gross_salary) AS total_gross_salary,
    SUM(s.net_salary) AS total_net_salary,
    s.status,
    MAX(s.created_at) AS created_at,
    MAX(s.approved_at) AS approved_at
FROM salaries s
JOIN projects p ON s.project_id = p.id
GROUP BY s.project_id, p.name, s.month, s.status;
```

## 💡 工资自动计算逻辑

### 生成工资时自动计算

```php
// 对于每个员工
foreach ($employees as $employee) {
    // 1. 基本工资
    $basicSalary = $employee->basic_salary;
    
    // 2. 读取考勤数据（缺勤扣款等）
    $attendanceDeduction = calculateAttendanceDeduction($employee, $month);
    
    // 3. 读取社保公积金（从 insurance_detail_records）
    $socialSecurity = getEmployeeSocialSecurity($employee, $month);
    $housingFund = getEmployeeHousingFund($employee, $month);
    
    // 4. 计算应发工资
    $grossSalary = $basicSalary + $allowance + $overtimePay + $bonus - $attendanceDeduction;
    
    // 5. 计算应纳税所得额
    $taxableIncome = $grossSalary - 5000 - $socialSecurity - $housingFund - $specialDeduction;
    
    // 6. 计算个税
    $personalTax = calculatePersonalTax($taxableIncome);
    
    // 7. 计算实发工资
    $netSalary = $grossSalary - $socialSecurity - $housingFund - $personalTax;
    
    // 8. 插入数据库
    Salary::create([
        'employee_id' => $employee->id,
        'project_id' => $projectId,
        'month' => $month,
        'basic_salary' => $basicSalary,
        'gross_salary' => $grossSalary,
        'social_security' => $socialSecurity,
        'housing_fund' => $housingFund,
        'personal_tax' => $personalTax,
        'net_salary' => $netSalary,
        'status' => 'draft'
    ]);
}
```
