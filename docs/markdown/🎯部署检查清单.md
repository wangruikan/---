## 🐛 问题排查

### 如果字段添加失败
```sql
-- 检查字段是否存在
SHOW COLUMNS FROM `projects` LIKE 'require_attendance';

-- 如果不存在，手动添加
ALTER TABLE `projects` 
ADD COLUMN `require_attendance` TINYINT(1) NOT NULL DEFAULT 1 
COMMENT '是否需要考勤：1-需要，0-不需要' 
AFTER `status`;
```

### 如果前端不显示
```bash
# 清除浏览器缓存
Ctrl + Shift + Delete

# 重启开发服务器
npm run dev
```

### 如果后端报错
```bash
# 清除所有缓存
cd backend
php artisan config:clear
php artisan cache:clear
php artisan route:clear
php artisan view:clear

# 重启PHP服务（如果使用PHP-FPM）
# 或重启整个Web服务器
```
