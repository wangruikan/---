# 🚀 资料交付管理 - 快速测试指南

## 📋 测试步骤

### 第1步：添加交付配置
1. 进入：**资料交付管理 → 交付配置**
2. 点击"添加配置"
3. 填写：
   - 选择项目
   - 交付周期：月度 或 季度
   - 交付方式：快递交付 或 电子推送
   - 资料清单：输入需要交付的文件（每行一个）
4. 点击"确定"

### 第2步：手动生成交付记录（用于测试）
1. 打开 Navicat 或其他数据库工具
2. 打开文件：`✅测试用-手动生成交付记录.sql`
3. **依次执行以下SQL：**

#### 👀 先查看配置是否存在
```sql
SELECT 
    id,
    project_id,
    delivery_cycle,
    delivery_method,
    account_set_id,
    is_active
FROM project_delivery_configs
WHERE is_active = 1;
```

#### 🎯 生成月度交付记录
```sql
INSERT INTO document_deliveries (
    config_id,
    project_id,
    account_set_id,
    delivery_cycle,
    delivery_method,
    delivery_period,
    status,
    handler_id,
    required_documents,
    created_at,
    updated_at
)
SELECT 
    id as config_id,
    project_id,
    account_set_id,
    delivery_cycle,
    delivery_method,
    DATE_FORMAT(NOW(), '%Y-%m') as delivery_period,
    'pending' as status,
    created_by as handler_id,
    required_documents,
    NOW() as created_at,
    NOW() as updated_at
FROM project_delivery_configs
WHERE is_active = 1 
  AND delivery_cycle = 'monthly';
```

#### 🎯 生成季度交付记录
```sql
INSERT INTO document_deliveries (
    config_id,
    project_id,
    account_set_id,
    delivery_cycle,
    delivery_method,
    delivery_period,
    status,
    handler_id,
    required_documents,
    created_at,
    updated_at
)
SELECT 
    id as config_id,
    project_id,
    account_set_id,
    delivery_cycle,
    delivery_method,
    CONCAT(YEAR(NOW()), '-Q', QUARTER(NOW())) as delivery_period,
    'pending' as status,
    created_by as handler_id,
    required_documents,
    NOW() as created_at,
    NOW() as updated_at
FROM project_delivery_configs
WHERE is_active = 1 
  AND delivery_cycle = 'quarterly';
```

#### ✅ 查看生成的记录
```sql
SELECT 
    dd.id,
    p.name as project_name,
    dd.delivery_cycle,
    dd.delivery_method,
    dd.delivery_period,
    dd.status,
    u.name as handler_name,
    dd.created_at
FROM document_deliveries dd
LEFT JOIN projects p ON dd.project_id = p.id
LEFT JOIN users u ON dd.handler_id = u.id
ORDER BY dd.created_at DESC;
```

### 第3步：查看交付记录
1. 刷新浏览器（Ctrl + F5）
2. 进入：**资料交付管理 → 交付记录**
3. 应该能看到刚刚生成的记录了！

### 第4步：测试交付功能

#### 📦 如果是"快递交付"：
1. 点击"提交快递"
2. 填写：
   - 快递单号
   - 实际交付资料（可修改）
3. 可以上传附件（可选）
4. 点击"提交"
5. 状态变为"已完成"

#### 💻 如果是"电子推送"：
1. 点击"提交电子资料"
2. 上传电子文件
3. 填写实际交付资料
4. 点击"提交"
5. 状态变为"已完成"

## 🗑️ 重新测试

TRUNCATE TABLE document_deliveries;

然后重复上面的步骤2-4。
