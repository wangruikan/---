### 2. 流程记录管理功能

#### 权限控制系统
**权限规则：**
- ✅ **超级管理员** - 完全访问权限
- ✅ **管理员** - 完全访问权限  
- ✅ **经理/高级经理** - 可以访问
- ✅ **审批级别2-4** - 可以访问
- ❌ **第1审批节点** - 无法访问
- ❌ **普通员工** - 无法访问

#### 后端实现
**文件：** `app/Http/Controllers/ProcessRecordController.php`

**API接口：**
- `GET /api/process-records` - 获取流程记录列表（带分页和筛选）
- `GET /api/process-records/stats` - 获取流程统计数据

**权限检查：**
```php
private function canViewProcessRecords($user)
{
    // 超级管理员检查
    if ($user->role === 'super_admin') return true;
    
    // 角色权限检查
    $allowedRoles = ['admin', 'manager', 'senior_manager'];
    if (in_array($user->role, $allowedRoles)) return true;
    
    // 审批级别检查
    if (isset($user->approval_level) && $user->approval_level >= 2) return true;
    
    return false;
}
```

const canViewProcessRecords = computed(() => {
  const user = userStore.userInfo
  if (!user) return false
  
  // 多层权限检查
  if (user.role === 'super_admin' || user.role === 'admin') return true
  if (['manager', 'senior_manager'].includes(user.role)) return true
  if (user.approval_level && user.approval_level >= 2) return true
  
  return false
})
