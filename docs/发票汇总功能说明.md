# 发票汇总功能说明

## 功能概述

发票汇总模块用于自动汇总所有审批通过的发票申请，方便财务人员统一管理和开票。

## 功能特点

- ✅ 自动汇总：发票申请审批通过后自动创建汇总记录
- ✅ 多维度筛选：支持日期、所属期、项目名称、状态筛选
- ✅ 完整信息：包含开票所需的所有信息
- ✅ 状态管理：支持待开票、已完成状态
- ✅ 快速操作：一键标记完成
- ✅ 数据导出：支持导出Excel

## 字段说明

| 字段 | 说明 | 计算方式 |
|------|------|----------|
| 序号 | 自动编号 | - |
| 所属期 | 发票所属月份 | 来自发票申请 |
| 单位名称 | 开票单位 | 来自发票申请 |
| 申请日期 | 发票申请日期 | 来自发票申请 |
| 开票方式 | 全额/差额/无 | 来自发票申请 |
| 开票种类 | 发票类型 | 来自发票申请 |
| 状态 | 待开票/已完成 | 系统管理 |
| 项目名称 | 关联项目 | 来自发票申请 |
| 开票金额 | 总金额 | 来自发票申请 |
| 扣除额 | 扣除金额 | 来自发票申请 |
| 税率 | 税率 | 来自发票申请 |
| 不含税金额 | 计算值 | (开票金额 - 扣除额) / (1 + 税率) |
| 开票税金 | 计算值 | 不含税金额 × 税率 |
| 税金 | 计算值 | 开票税金 |
| 开票日期 | 实际开票日期 | 手动填写 |
| 是否完成 | 完成标识 | 手动标记 |
| 开票人 | 开票人员 | 手动填写 |
| 发票号码 | 发票编号 | 手动填写 |
| 备注 | 备注信息 | 来自发票申请或手动填写 |

## 使用流程

### 1. 自动创建汇总记录

当发票申请审批通过后，系统会自动：
1. 创建发票汇总记录
2. 计算不含税金额、开票税金等
3. 状态设置为"待开票"

### 2. 查询和筛选

支持以下筛选条件：
- **申请日期范围**：选择开始和结束日期
- **所属期**：选择月份
- **项目名称**：输入项目名称关键词
- **状态**：待开票/已完成

### 3. 编辑记录

点击"编辑"按钮可以修改：
- 开票日期
- 开票人
- 发票号码
- 状态
- 是否完成
- 备注

### 4. 标记完成

对于待开票的记录，点击"标记完成"按钮：
1. 输入发票号码
2. 系统自动设置开票日期为当天
3. 状态更新为"已完成"

### 5. 导出Excel

点击"导出Excel"按钮可以导出当前筛选条件下的所有记录。

## 安装步骤

### 1. 运行数据库迁移

```bash
php artisan migrate --path=database/migrations/2025_12_27_000001_create_invoice_summaries_table.php
```

### 2. 添加前端路由

在 `src/router/index.js` 中添加路由：

```javascript
{
  path: '/invoice-summary',
  name: 'InvoiceSummary',
  component: () => import('@/views/InvoiceSummary/index.vue'),
  meta: { title: '发票汇总', requiresAuth: true }
}
```

### 3. 添加菜单

在系统菜单中添加"发票汇总"菜单项，链接到 `/invoice-summary`。

### 4. 配置权限（可选）

如果需要权限控制，在权限管理中添加：
- `invoice_summaries.view` - 查看发票汇总
- `invoice_summaries.edit` - 编辑发票汇总
- `invoice_summaries.export` - 导出发票汇总

## API 接口

### 获取列表

```
GET /api/invoice-summaries
```

参数：
- `page`: 页码
- `per_page`: 每页数量
- `start_date`: 开始日期
- `end_date`: 结束日期
- `period`: 所属期
- `project_name`: 项目名称
- `status`: 状态

### 更新记录

```
PUT /api/invoice-summaries/{id}
```

参数：
- `invoice_date`: 开票日期
- `invoicer`: 开票人
- `invoice_number`: 发票号码
- `status`: 状态
- `is_completed`: 是否完成
- `remarks`: 备注

### 标记为已完成

```
POST /api/invoice-summaries/{id}/mark-completed
```

参数：
- `invoice_number`: 发票号码
- `invoice_date`: 开票日期（可选）
- `invoicer`: 开票人（可选）

### 导出Excel

```
GET /api/invoice-summaries/export
```

参数：同获取列表

## 注意事项

1. **自动创建**：只有审批通过的发票申请才会创建汇总记录
2. **计算准确性**：不含税金额和税金会自动计算，请确保发票申请中的数据准确
3. **状态管理**：标记完成后状态会自动更新，无需手动修改
4. **数据关联**：汇总记录与发票申请关联，删除发票申请会级联删除汇总记录
