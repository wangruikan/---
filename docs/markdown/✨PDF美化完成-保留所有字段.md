## 兼容性说明

### mpdf可能的问题

1. **渐变效果**
   - 如果mpdf不支持渐变，会显示为纯色
   - 备选方案：改为纯色 #667eea

2. **阴影效果**
   - 可能不显示
   - 不影响阅读，只是装饰

3. **圆角**
   - 可能不显示
   - 不影响整体效果

### 如果需要调整

#### 改为纯色背景
```css
.label-cell {
    background: #667eea;  /* 纯色代替渐变 */
    color: #fff;
}
```

#### 调整颜色深浅
```css
.label-cell {
    background: #5a67d8;  /* 更深 */
    /* 或 */
    background: #7c8df8;  /* 更浅 */
}
```

#### 改变配色方案
```css
/* 蓝绿配色 */
background: linear-gradient(135deg, #00b4d8 0%, #0077b6 100%);

/* 橙红配色 */
background: linear-gradient(135deg, #ff6b6b 0%, #ee5a6f 100%);

/* 绿色配色 */
background: linear-gradient(135deg, #51cf66 0%, #37b24d 100%);
```
