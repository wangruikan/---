## 📊 数据流程

1. **创建开票任务** → 业务人员填写"项目"字段
2. **保存到数据库** → `invoice_applications.project_name` 字段
3. **查看详情** → 在详情页面的基本信息中显示项目名称
4. **生成Excel** → 从 `application.project_name` 读取并填充到Excel第2行

---

## 🎯 功能说明

### 区别说明

| 字段位置 | 字段名 | 用途 | 来源 |
|---------|--------|------|------|
| `invoice_applications.project_name` | 项目名称 | 整个发票申请的项目归属 | 业务人员在创建时填写 |
| `invoice_items.project_name` | 明细项目 | 每个扣除明细项的项目分类 | 从发票项目配置中选择 |

### 生成Excel时的项目字段

- **修改前**：从第一个明细项获取 `$application->items->first()->project_name`
- **修改后**：从发票申请本身获取 `$application->project_name`
- **原因**：每个明细项可能属于不同的项目分类，而整个发票申请应该有一个统一的项目归属
