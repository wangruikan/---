# ✅ 新增员工 - 示例填充功能

## 功能说明

在新增员工页面添加了**示例填充按钮**，一键填充测试数据，包括基数信息。

## 使用方法

### 步骤 1️⃣：打开新增员工对话框
1. 进入"人员档案管理"页面
2. 点击"新增员工"按钮
3. 打开新增员工对话框

### 步骤 2️⃣：点击示例填充按钮
在新增员工对话框中，点击**"示例填充"**按钮（位于"所属项目"字段下方）

![示例填充按钮位置]
```
┌─────────────────────────────────┐
│ 新增员工                         │
├─────────────────────────────────┤
│ 工号：EMP001                     │
│ 姓名：张三                       │
│ ...                             │
│ 所属项目：[自动选择第一个项目]   │
│                                 │
│              [示例填充] ⭐       │
│                                 │
└─────────────────────────────────┘
```

### 步骤 3️⃣：确认填充
系统会弹出确认对话框，点击"确定"即可填充示例数据

```
┌─────────────────────────────────┐
│ 示例填充确认                     │
├─────────────────────────────────┤
│ 确定要填充示例数据吗？          │
│ 这将覆盖当前表单中的数据。      │
│                                 │
│ [取消]  [确定]                  │
└─────────────────────────────────┘
```

### 步骤 4️⃣：查看填充结果
填充完成后，表单中会自动填入以下数据：

## 📝 注意事项

1. **项目选择**：系统会自动选择第一个可用的项目
   - 如果没有项目，则不选择任何项目
   - 可以手动修改项目选择

2. **基数值**：示例基数为 5000
   - 可以根据实际需要修改
   - 建议使用合理的基数值进行测试

3. **身份证号**：示例身份证号为虚拟数据
   - 仅用于测试目的
   - 实际使用时应替换为真实身份证号

4. **数据覆盖**：点击示例填充会覆盖当前表单数据
   - 系统会弹出确认对话框
   - 用户需要确认才能执行

5. **修改权限**：查看模式下示例填充按钮被禁用
   - 只有在新增或编辑模式下才能使用
   - 查看模式下所有字段都是只读的
