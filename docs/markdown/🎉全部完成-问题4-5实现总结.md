## 📝 使用说明

### 小程序端 - 手写签名

1. **填写入职登记表**
   - 填写所有基本信息
   - 滚动到底部"声明与签名"区域

2. **手写签名**
   - 在空白画布区域用手指书写签名
   - 签名完成后自动保存为图片
   - 可以点击"清除签名"重新签名

3. **查看签名**
   - 签名后会显示签名预览图
   - 可以点击"重新签名"进行修改

4. **提交**
   - 必须完成签名才能提交
   - 提交后签名永久保存

### PC端 - 查看签名

1. **查看员工信息**
   - 进入"人员档案" → 选择员工 → 点击"查看"
   - 切换到"个人信息"标签页
   - 滚动到底部查看"本人签名"

2. **签名显示**
   - 已签名：显示签名图片
   - 未签名：显示"该员工尚未签名"

### PC端 - 批量导出PDF

1. **选择员工**
   - 在员工列表中勾选需要导出的员工
   - 可以勾选一个或多个

2. **导出PDF**
   - 点击"批量导出PDF (N)"按钮
   - 系统提示"正在生成PDF，请稍候..."

3. **下载结果**
   - **单个员工**: 直接下载PDF文件（例如：张三_入职登记表.pdf）
   - **多个员工**: 下载ZIP压缩包（例如：入职登记表_1731312345678.zip）

4. **查看PDF**
   - 解压ZIP（如果是多个员工）
   - 打开PDF文件查看完整的入职登记表
   - 包含所有信息和签名

## ⚠️ 注意事项

### 签名功能
1. **文件大小**: base64签名图片约50-100KB，TEXT字段足够
2. **兼容性**: 支持微信小程序、H5等平台
3. **清晰度**: canvas尺寸400rpx，确保签名清晰
4. **背景**: 签名图片无背景，便于显示

### PDF导出功能
1. **依赖**: 需要安装dompdf/dompdf库
2. **字体**: 使用DejaVu Sans支持中文
3. **图片**: base64签名图片直接嵌入PDF
4. **性能**: 
   - 单个PDF生成约1-2秒
   - 批量10个员工约10-15秒
   - 建议一次不超过50个员工
5. **存储**: 临时文件自动删除
6. **权限**: 确保storage/app/temp目录可写

### 错误处理
1. **员工未填表**: 提示"该员工未填写入职登记表"
2. **签名缺失**: PDF显示空白签名位
3. **生成失败**: 记录日志，跳过该员工继续处理其他
4. **网络超时**: 前端设置合理的超时时间
