撤回按钮显示条件：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 必须在"我发起的"标签页
✓ 审批状态为"待审批"（pending）
✓ 第二个审批人还未审批
✓ 流程在第二步或之前

点击撤回按钮后：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 弹出确认对话框：
   ┌─────────────────────────────────────┐
   │  ⚠️  撤回确认                        │
   ├─────────────────────────────────────┤
   │  确定要撤回此审批吗？                │
   │  撤回后审批流程将终止。              │
   │                                      │
   │  [取消]           [确定撤回]        │
   └─────────────────────────────────────┘

2. 确认后：
   - 审批状态变为"已撤回"
   - 所有待审批记录标记为"已撤回"
   - 业务数据状态更新
   - 显示成功提示："审批已撤回"
