# 商业险保单统计功能

## 功能说明

在"商业险退保管理"页面新增了"保单统计"标签页，用于统计每个商业险保单在每个月份的参保人数和总金额。

## 功能特点

### 1. 数据来源
- 从 `insurance_personnel` 表中读取参保人员数据
- 解析 `other_insurance_policies` 字段获取商业险信息
- **只统计保险类型为"商业险"的保单**

### 2. 统计维度
- **按保单分组**：每个保单独立展示
- **按月份统计**：显示每个月份的数据
- **只显示有数据的月份**：没有数据的月份不会显示

### 3. 统计指标
- 参保人数
- 总金额
- 人均金额（自动计算）

### 4. 筛选功能
- **保单筛选**：可选择特定保单查看
- **月份范围筛选**：可指定开始月份和结束月份

### 5. 明细查看
- 点击"查看明细"可查看该月份的所有参保员工
- 显示员工姓名、员工ID、金额

## 使用方法

### 访问入口
系统管理 → 商业险退保管理 → 保单统计（标签页）

### 操作步骤
1. 切换到"保单统计"标签页
2. （可选）选择要查看的保单
3. （可选）选择月份范围
4. 点击"查询"按钮
5. 展开保单查看月度统计数据
6. 点击"查看明细"查看具体员工列表

## API接口

### 请求地址
```
GET /api/insurance-surrenders/policy-statistics
```

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| account_set_id | integer | 是 | 账套ID |
| policy_id | integer | 否 | 保单ID（筛选特定保单） |
| start_month | string | 否 | 开始月份（格式：YYYY-MM） |
| end_month | string | 否 | 结束月份（格式：YYYY-MM） |

## 技术实现

### 后端
- **控制器**：`App\Http\Controllers\InsuranceSurrenderController@getPolicyStatistics`
- **路由**：`routes/api.php` 中的 `insurance-surrenders/policy-statistics`
- **权限**：复用 `insurance_surrender.view` 权限

### 前端
- **页面**：`src/views/InsuranceSurrender/index.vue`
- **组件**：使用 Element Plus 的 Tabs、Collapse、Table 等组件
- **状态管理**：使用 Vue 3 Composition API

## 注意事项

1. **只统计商业险**：系统会自动过滤掉非商业险类型的保单
2. **月份判断**：使用 `insurance_personnel` 表的 `last_updated_at` 或 `created_at` 字段确定月份
3. **数据准确性**：统计数据基于参保人员表的实时数据，确保数据的准确性
4. **性能优化**：建议使用筛选功能减少数据量，提高查询速度
