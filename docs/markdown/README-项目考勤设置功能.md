# 项目考勤设置功能

## 🎯 功能说明

在项目设置中添加"是否需要考勤"开关，灵活控制工资表生成的前置条件。

- **需要考勤**：必须先审批考勤表才能生成工资表
- **无需考勤**：可以直接生成工资表

## 💡 使用场景

### 场景1：正式员工项目
- 设置：需要考勤 = ✅ 开启
- 效果：必须先审批考勤表才能生成工资表

### 场景2：外包项目
- 设置：需要考勤 = ❌ 关闭
- 效果：可以直接生成工资表，无需考勤审批

## 📊 工资表生成规则

| 项目设置 | 考勤状态 | 能否生成工资表 | 显示标签 |
|---------|---------|--------------|---------|
| 需要考勤 | 已审批 | ✅ 可以 | 项目名称 |
| 需要考勤 | 未审批 | ❌ 不可以 | 项目名称 (考勤未审批) |
| 无需考勤 | - | ✅ 可以 | 项目名称 (无需考勤) |

## ❓ 常见问题

**Q: 修改设置后，已有的工资表会变吗？**  
A: 不会。只影响新生成的工资表。

**Q: 默认新项目需要考勤吗？**  
A: 是的。默认需要考勤。

**Q: 无需考勤的项目还能创建考勤表吗？**  
A: 可以。考勤设置只影响工资表生成条件。
