# 付款附件上传组件 - 使用说明

## 🎉 新组件完成

已创建 `PaymentAttachmentUploader.vue` 组件，集成了**发票上传**、**其他附件上传**和**签名盖章**功能，可在多个地方复用。

## 📦 组件文件

- **组件路径**: `src/components/PaymentAttachmentUploader.vue`
- **依赖组件**:
  - `InvoiceUploader.vue` - 发票上传
  - `FormToWordGenerator.vue` - 表格生成PDF
  - `PDFSignatureEditor.vue` - PDF签名盖章

## 🎯 使用方法

### 1. 基本用法

```vue
<template>
  <PaymentAttachmentUploader
    ref="uploaderRef"
    v-model:invoice-file-list="invoiceFiles"
    v-model:other-file-list="otherFiles"
  />
</template>

<script setup>
import { ref } from 'vue'
import PaymentAttachmentUploader from '@/components/PaymentAttachmentUploader.vue'

const uploaderRef = ref(null)
const invoiceFiles = ref([])
const otherFiles = ref([])
</script>
```

### 2. 完整配置

```vue
<PaymentAttachmentUploader
  ref="paymentAttachmentUploaderRef"
  v-model:invoice-file-list="invoiceFileList"
  v-model:other-file-list="paymentFileList"
  :invoice-limit="10"
  :other-limit="5"
  :show-form-generator="true"
  form-button-text="填写表格生成PDF"
  form-title="工资付款申请表"
  @files-changed="handleFilesChanged"
/>
```

## 📋 Props 属性

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `invoiceFileList` | Array | `[]` | 发票文件列表（v-model） |
| `otherFileList` | Array | `[]` | 其他附件列表（v-model） |
| `invoiceLimit` | Number | `10` | 发票上传数量限制 |
| `otherLimit` | Number | `5` | 其他附件数量限制 |
| `showFormGenerator` | Boolean | `true` | 是否显示表格生成器按钮 |
| `formButtonText` | String | `'填写表格生成PDF'` | 表格生成器按钮文本 |
| `formTitle` | String | `'付款申请表'` | 表格生成器标题 |

## 📤 Events 事件

### `files-changed`

文件列表变化时触发

```javascript
@files-changed="handleFilesChanged"

// 事件参数
{
  invoiceFiles: [],    // 发票文件列表
  otherFiles: [],      // 其他附件列表
  allFiles: []         // 所有文件列表
}
```

## 🔧 暴露方法

通过 `ref` 可以调用以下方法：

### `clearAll()`
清空所有文件（发票+其他附件）

```javascript
uploaderRef.value.clearAll()
```

### `clearInvoices()`
清空发票文件

```javascript
uploaderRef.value.clearInvoices()
```

### `clearOthers()`
清空其他附件

```javascript
uploaderRef.value.clearOthers()
```

### `getAllFiles()`
获取所有文件

```javascript
const allFiles = uploaderRef.value.getAllFiles()
```

### `getInvoiceFiles()`
获取发票文件

```javascript
const invoices = uploaderRef.value.getInvoiceFiles()
```

### `getOtherFiles()`
获取其他附件

```javascript
const others = uploaderRef.value.getOtherFiles()
```

## ✨ 功能特性

### 1. **发票上传**
- ✅ 支持PDF、JPG、PNG、GIF格式
- ✅ 单个文件最大50MB
- ✅ 可自定义上传数量限制
- ✅ 自动校验文件格式和大小

### 2. **其他附件上传**
- ✅ 支持PDF、Word、Excel、图片等格式
- ✅ 单个文件最大50MB
- ✅ 可自定义上传数量限制

### 3. **表格生成PDF**
- ✅ 可选功能，可以通过 `showFormGenerator` 控制显示
- ✅ 支持"情况说明单"和"付款申请单"两种表格
- ✅ 生成的PDF自动添加到其他附件列表

### 4. **签名盖章**
- ✅ 自动识别所有PDF文件（发票+其他附件）
- ✅ 支持选择要签名的文件
- ✅ 区分显示发票和其他附件（不同标签颜色）
- ✅ 签名后自动替换原文件
- ✅ 文件名自动添加"_已签名_时间戳"

## 📝 使用示例

### 示例1：工资管理页面

```vue
<template>
  <el-dialog v-model="dialogVisible" title="发起付款">
    <el-form>
      <!-- 其他表单项 -->
      
      <!-- 使用组件 -->
      <PaymentAttachmentUploader
        ref="uploaderRef"
        v-model:invoice-file-list="invoiceFiles"
        v-model:other-file-list="otherFiles"
        :invoice-limit="10"
        :other-limit="5"
        form-title="工资付款申请表"
      />
    </el-form>
    
    <template #footer>
      <el-button @click="handleSubmit">提交</el-button>
    </template>
  </el-dialog>
</template>

<script setup>
import { ref } from 'vue'
import PaymentAttachmentUploader from '@/components/PaymentAttachmentUploader.vue'

const dialogVisible = ref(false)
const uploaderRef = ref(null)
const invoiceFiles = ref([])
const otherFiles = ref([])

const handleSubmit = async () => {
  // 获取所有文件
  const allFiles = uploaderRef.value.getAllFiles()
  
  // 合并上传
  for (const file of allFiles) {
    await uploadFile(file)
  }
  
  // 清空文件列表
  uploaderRef.value.clearAll()
}
</script>
```

## 📌 注意事项

1. **文件格式校验**
   - 发票只支持PDF和图片格式
   - 其他附件支持更多格式

2. **文件大小限制**
   - 单个文件不超过50MB
   - 超过限制会自动提示并拒绝

3. **签名盖章**
   - 只能对PDF文件进行签名
   - 签名后的文件会替换原文件
   - 原文件名会添加"_已签名"后缀

4. **v-model双向绑定**
   - 必须使用 `v-model:invoice-file-list` 和 `v-model:other-file-list`
   - 不能简写为 `v-model`
