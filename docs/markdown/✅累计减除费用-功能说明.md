# 累计减除费用 - 功能说明

## 📝 功能概述

**累计减除费用** = 5000 × 实际工作月数

这个字段用于计算个人所得税，是国家规定的每月基本减除费用（起征点）的累计值。

---

## 🎯 计算规则

### 基本公式
```
累计减除费用 = 5000 × 实际工作月数
```

### 月数计算规则

#### 1️⃣ 入职时间在本年1月或之前
**实际工作月数 = 当前月份**

**示例**：
- 入职时间：2023-06-15（去年入职）
- 当前月份：2025-09（9月）
- 实际工作月数：9个月
- 累计减除费用：5000 × 9 = **45,000元**

#### 2️⃣ 入职时间在本年1月之后
**实际工作月数 = 当前月份 - 入职月份 + 1**

**示例**：
- 入职时间：2025-05-15（今年5月入职）
- 当前月份：2025-09（9月）
- 实际工作月数：9 - 5 + 1 = 5个月
- 累计减除费用：5000 × 5 = **25,000元**

#### 3️⃣ 入职时间在本年之后
**实际工作月数 = 0**（这种情况不应该发生，但做保护处理）

---

## 💡 具体案例

### 案例1：老员工（入职时间早于本年）
```
入职时间：2024-03-10
计算2025年9月工资：
- 实际工作月数 = 9（1月到9月）
- 累计减除费用 = 5000 × 9 = 45,000元
```

### 案例2：本年1月入职
```
入职时间：2025-01-05
计算2025年9月工资：
- 实际工作月数 = 9（1月到9月）
- 累计减除费用 = 5000 × 9 = 45,000元
```

### 案例3：本年5月入职
```
入职时间：2025-05-15
计算2025年9月工资：
- 实际工作月数 = 9 - 5 + 1 = 5（5月到9月）
- 累计减除费用 = 5000 × 5 = 25,000元
```

### 案例4：本年9月入职
```

计算2025年9月工资：
- 实际工作月数 = 9 - 9 + 1 = 1（仅9月）
- 累计减除费用 = 5000 × 1 = 5,000元
```

## 🔧 实现细节

### 数据库字段
```sql
ALTER TABLE `salaries` 
ADD COLUMN `cumulative_basic_deduction` DECIMAL(10, 2) DEFAULT 0 
COMMENT '累计减除费用（5000×月数）' 
AFTER `cumulative_income`;
```

### 后端逻辑
**文件**: `app/Http/Controllers/SalaryController.php`

```php
// 累计减除费用 = 5000 × 实际工作月数
// 需要考虑员工入职时间：如果入职时间在当年1月之后，则从入职月份开始计算
$actualMonthCount = $monthCount; // 默认从1月开始

if ($employee->hire_date) {
    $hireDate = \Carbon\Carbon::parse($employee->hire_date);
    $currentYear = intval($year);
    
    // 如果入职年份是当年，且入职月份在1月之后
    if ($hireDate->year == $currentYear && $hireDate->month > 1) {
        // 从入职月份到本月的月数
        $actualMonthCount = $monthCount - $hireDate->month + 1;
    }
    // 如果入职年份晚于当年，则月数为0（不应该发生，但做保护）
    elseif ($hireDate->year > $currentYear) {
        $actualMonthCount = 0;
    }
}

$cumulativeBasicDeduction = 5000 * $actualMonthCount;
```

## 📊 在个税计算中的应用

累计减除费用是计算累计应纳税所得额的重要组成部分：

```
累计应纳税所得额 = 累计收入 
                - 累计减除费用（5000×月数）
                - 累计社保公积金
                - 累计专项附加扣除
```

**示例**：
```
累计收入：100,000元
累计减除费用：45,000元（5000×9）
累计社保公积金：10,000元
累计专项附加扣除：5,000元

累计应纳税所得额 = 100,000 - 45,000 - 10,000 - 5,000 = 40,000元
```
