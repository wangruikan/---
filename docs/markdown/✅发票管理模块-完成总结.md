## 📊 特色功能

### 1. Excel自动生成 ⭐
- ✅ 使用 PhpSpreadsheet 库
- ✅ 自动格式化（边框、对齐、字体）
- ✅ 千分位金额格式

**Excel格式：**
```
┌─────────────────────────────────┐
│        扣除明细表                │
├────┬─────────┬────────┬─────────┤
│序号│ 名称     │ 金额    │ 备注     │

│ 12 │商业险    │3,000   │         │
├────┼─────────┼────────┼─────────┤
│    │合计      │108,000 │         │
└────┴─────────┴────────┴─────────┘
```

### 2. 红冲机制 ⭐
- ✅ 驳回后可以重新提交
- ✅ 重新提交时：
  - 原申请状态变为"红冲"
  - 创建新申请（复制所有数据）
  - 新申请生成新的申请单号
  - 自动提交审批（跳过第1节点）
- ✅ 红冲申请保留在列表中（用于追踪历史）

### 3. 审批流程跳过第1节点 ⭐
- ✅ 业务人员（经办）发起申请
- ✅ 提交时不需要第1节点审批（自动跳过）
- ✅ 直接从第2节点开始审批流程
- ✅ 节省审批时间
