参保明细接口 (`insurance-changes/details`) 返回的字段：

这些值是**在接口中计算出来的**，不是直接从数据库字段读取的！

## 📊 参保明细接口的计算逻辑

**文件：** `app/Http/Controllers/InsuranceChangeController.php`  
**方法：** `generatePersonnelDetail()`  
**位置：** 第 514-521 行

```php
if ($paymentCycle === 'year') {
    // 按年付款：正常数据的大额为0
    $largeMedicalCompanyAmount = 0;
    $largeMedicalEmployeeAmount = 0;
} else {
    // 按月付款：正常计算
    if ($largeMedicalConfig['calculation_type'] === 'base') {
        // ✅ 按基数和比例计算
        $largeMedicalCompanyAmount = $largeMedicalBase * floatval($largeMedicalConfig['company_ratio']);
        $largeMedicalEmployeeAmount = $largeMedicalBase * floatval($largeMedicalConfig['employee_ratio']);
    } else {
        // ✅ 按固定金额
        $largeMedicalCompanyAmount = floatval($largeMedicalConfig['company_amount']);
        $largeMedicalEmployeeAmount = floatval($largeMedicalConfig['employee_amount']);
    }
}
```

### 计算逻辑说明

1. **检查付款周期**：
   - `payment_cycle` === 'year' → 金额为 0
   - `payment_cycle` === 'month' → 继续计算

2. **检查计算类型**：
   - `calculation_type` === 'base' → 基数 × 比例
   - 否则 → 使用固定金额 (`company_amount` 和 `employee_amount`)

// 大额医疗（如果有）- 从 large_medical_insurance_config JSON 字段读取
$largeMedicalCompanyAmount = 0;

    $largeMedicalConfig = is_string($insurancePersonnel->large_medical_insurance_config)
        ? json_decode($insurancePersonnel->large_medical_insurance_config, true)

        $paymentCycle = $largeMedicalConfig['payment_cycle'] ?? 'month';
        
        if ($paymentCycle === 'year') {
            // 按年付款：正常数据的大额为0
            $largeMedicalCompanyAmount = 0;
            $largeMedicalEmployeeAmount = 0;
        } else {
            // 按月付款：正常计算
            if ($largeMedicalConfig['calculation_type'] === 'base') {
                // 按基数计算
                $largeMedicalCompanyAmount = $largeMedicalBase * floatval($largeMedicalConfig['company_ratio']);
                $largeMedicalEmployeeAmount = $largeMedicalBase * floatval($largeMedicalConfig['employee_ratio']);
            } else {
                // 按固定金额
                $largeMedicalCompanyAmount = floatval($largeMedicalConfig['company_amount']);
                $largeMedicalEmployeeAmount = floatval($largeMedicalConfig['employee_amount']);
            }
        }
        
        $details['large_medical'] = [
            'name' => '大额医疗',
            'base' => $largeMedicalBase,
            'company_amount' => round($largeMedicalCompanyAmount, 2),
            'personal_amount' => round($largeMedicalEmployeeAmount, 2),
        ];
    }
}
