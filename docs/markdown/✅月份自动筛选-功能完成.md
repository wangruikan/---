# ✅ 月份自动筛选 - 功能完成

## 🎯 功能说明

在参保增减页面，当更改月份筛选时，系统会**自动触发查询**，无需手动点击"查询"按钮。

## ✅ 实现方式

### 添加 `watch` 监听器

```javascript
// 监听月份筛选变化，自动触发查询
watch(() => detailFilterForm.value.month, (newMonth, oldMonth) => {
  // 确保月份发生了变化，且不是初始化时的变化
  if (newMonth !== oldMonth && oldMonth !== undefined) {
    // 根据当前激活的标签页触发相应的查询
    if (activeTab.value === 'details') {
      if (detailActiveTab.value === 'social') {
        loadDetails()  // 社保明细
      } else if (detailActiveTab.value === 'compensation') {
        loadCompensationDetails()  // 社保补交明细
      } else if (detailActiveTab.value === 'housingFundCompensation') {
        loadHousingFundCompensationDetails()  // 公积金补交明细
      }
    } else if (activeTab.value === 'summaries') {
      loadSummaries()  // 汇总表
    }
  }
})
```

## 🔍 监听逻辑

### 1. 监听条件

```javascript
if (newMonth !== oldMonth && oldMonth !== undefined)
```

- `newMonth !== oldMonth`: 月份确实发生了变化
- `oldMonth !== undefined`: 不是第一次初始化（避免页面加载时触发）

### 2. 根据当前标签触发

```javascript
if (activeTab.value === 'details') {
  // 在"参保明细"主标签下
  if (detailActiveTab.value === 'social') {
    loadDetails()  // 社保明细子标签
  } else if (detailActiveTab.value === 'compensation') {
    loadCompensationDetails()  // 社保补交明细子标签
  } else if (detailActiveTab.value === 'housingFundCompensation') {
    loadHousingFundCompensationDetails()  // 公积金补交明细子标签
  }
} else if (activeTab.value === 'summaries') {
  // 在"汇总表"主标签下
  loadSummaries()
}
```

## 💡 避免重复查询

### 初始化时不触发

```javascript
if (oldMonth !== undefined)
```

- 页面首次加载时，`oldMonth` 是 `undefined`
- 此时不触发查询，避免重复请求
- 只有用户手动更改月份时才触发

### 相同月份不触发

```javascript
if (newMonth !== oldMonth)
```

- 只有当月份真正改变时才触发
- 避免不必要的API请求
