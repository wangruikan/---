2. 在侧边栏找到"投标项目管理"

## 📊 典型使用场景

### 场景1：收到新的招标信息
1. 点击"新建项目"
2. 填写项目名称、招标单位
3. 设置投标截止时间
4. 保存后系统自动生成项目编号

### 场景2：准备投标资料
1. 在项目详情切换到"文档管理"
2. 上传招标文件
3. 上传投标文件、技术方案、报价单
4. 系统自动记录上传时间和人员

### 场景3：投标过程跟踪
1. 提交投标：更新状态为"已提交"
2. 开标：更新状态为"已开标"，添加进度记录
3. 评标：更新状态为"评标中"
4. 结果公布：设置投标结果

### 场景4：中标后管理
1. 设置投标结果为"中标"
2. 填写中标金额和日期
3. 更新状态为"已签约"
4. 上传合同文件
5. 项目执行完成后更新为"已完成"

## 🔥 实用技巧

### 技巧1：快速查找项目
使用搜索框输入关键词，支持：
- 项目名称
- 项目编号
- 招标单位名称

### 技巧2：批量筛选
同时使用多个筛选条件：
- 项目类别 + 项目状态
- 投标结果 + 时间范围

### 技巧3：关注即将到期
- 查看统计卡片中的"即将到期"数量
- 在列表中会以黄色标签提示
- 建议每天登录查看

### 技巧4：善用进度记录
重要节点及时添加记录：
- 收到招标文件
- 参加现场勘察
- 提交投标文件
- 参加开标会议
- 结果公布

### 技巧5：文档版本管理
同一文档多次修改时：
- 上传时填写版本号（如1.0、1.1、2.0）
- 旧版本可以保留或删除
- 便于追溯修改历史

## 📋 字段说明速查

### 必填字段
- **项目名称**：投标项目的完整名称

### 重要字段
- **项目类别**：如劳务派遣、物业管理、保洁服务等

- **备注**：其他需要说明的信息

## ❓ 常见问题快速解答

**Q: 项目编号可以自定义吗？**  
A: 不可以，系统自动生成（格式：BID+日期+序号），确保唯一性。

**Q: 可以修改已设置的投标结果吗？**  
A: 可以重新设置，但建议谨慎操作，每次设置都会记录日志。

**Q: 删除项目会同时删除文档吗？**  
A: 是的，删除项目会删除所有关联数据且不可恢复，请谨慎。

**Q: 支持哪些文件格式？**  
A: 支持所有常见格式（PDF、Word、Excel、图片等），单个文件最大50MB。

**Q: 如何导出项目数据？**  
A: 当前版本暂不支持导出，后续版本会添加Excel导出功能。

**Q: 可以设置提醒吗？**  
A: 系统会自动检测即将到期的项目（3天内），后续版本会添加自定义提醒。

**Q: 多个人可以同时编辑一个项目吗？**  
A: 可以，但建议分工明确，避免数据冲突。

**Q: 历史项目如何归档？**  
A: 将状态更新为"已完成"或"已取消"，可通过状态筛选查看。
