# ✅ 多级表头 - (X月)单位部分标题

## 📋 需求

在保险明细上方添加一个"(X月)单位部分"的大标题，类似Excel的合并单元格效果。

## 💻 核心实现

### 1. 多级表头结构

```vue
<!-- (X月)单位部分 - 大标题（父级列）-->
<el-table-column :label="getCurrentMonthLabel()" align="center">
  <!-- 子列1：养老保险 -->
  <el-table-column label="养老保险" width="100" align="right">
    <template #default="{ row }">
      {{ getInsuranceValue(row, '养老保险', 'company_amount') }}
    </template>
  </el-table-column>
  
  <!-- 子列2：医疗保险 -->
  <el-table-column label="医疗保险" width="100" align="right">
    <template #default="{ row }">
      {{ getInsuranceValue(row, '医疗保险', 'company_amount') }}
    </template>
  </el-table-column>
  
  <!-- ... 更多子列 ... -->
</el-table-column>
```

**说明**：
- 父级 `el-table-column`：显示"(X月)单位部分"
- 子级 `el-table-column`：显示具体的险种和基数
- Element Plus 会自动渲染为多级表头

### 2. 动态月份标签

```javascript
// 获取当前月份标签（用于表头）
const getCurrentMonthLabel = () => {
  if (!currentSheet.value || !currentSheet.value.month) return '(本月)单位部分'
  
  const month = currentSheet.value.month // 格式：2025-10
  const monthNumber = month.split('-')[1] // 提取月份：10
  
  return `(${parseInt(monthNumber)}月)单位部分`
}
```

**功能**：
- 从工资表的 `month` 字段提取月份
- 格式化为 "(10月)单位部分"
- 如果没有月份信息，默认显示 "(本月)单位部分"

## 🔧 Element Plus 多级表头说明

### 基本语法

```vue
<!-- 父级列 -->
<el-table-column label="父级标题" align="center">
  <!-- 子列1 -->
  <el-table-column label="子列1" width="100">
    <template #default="{ row }">
      {{ row.value1 }}
    </template>
  </el-table-column>
  
  <!-- 子列2 -->
  <el-table-column label="子列2" width="100">
    <template #default="{ row }">
      {{ row.value2 }}
    </template>
  </el-table-column>
</el-table-column>
```

### 渲染效果

```
┌──────────────────────┐
│      父级标题        │
├──────────┬───────────┤
│  子列1   │   子列2   │
├──────────┼───────────┤
│  数据1   │   数据2   │
└──────────┴───────────┘
```

### 多层嵌套

```vue
<!-- 第一层 -->
<el-table-column label="一级标题">
  <!-- 第二层 -->
  <el-table-column label="二级标题">
    <!-- 第三层 -->
    <el-table-column label="三级标题" width="100">
      <template #default="{ row }">
        {{ row.value }}
      </template>
    </el-table-column>
  </el-table-column>
</el-table-column>
```
