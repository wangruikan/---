## 🔄 合并文件逻辑

两个页面都使用相同的文件合并逻辑：

```javascript
// 合并发票和其他附件
const allFiles = [...invoiceFileList.value, ...paymentFileList.value]

// 上传所有文件
for (const file of allFiles) {
  await uploadFile(file)
}

// 显示统计信息
const message = `已上传 ${uploadCount}/${allFiles.length} 个文件（发票 ${invoiceFileList.value.length} 个，其他附件 ${paymentFileList.value.length} 个）`
```
