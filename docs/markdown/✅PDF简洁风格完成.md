## 颜色说明

### #e8f4f8（淡蓝色）
这是标签单元格的背景色：
- 非常淡的蓝色
- 不刺眼
- 与白色协调
- 清晰区分标签和内容

### 如需调整颜色

更淡：
```css
background-color: #f0f8ff;  /* 更淡的蓝 */
```

稍深：
```css
background-color: #d4e9f7;  /* 稍深的蓝 */
```

改为灰色：
```css
background-color: #f5f5f5;  /* 浅灰 */
```
