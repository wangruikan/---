## 📝 响应格式规则

### 在本项目中的标准用法

```javascript
// ✅ 正确写法
const response = await someApi()

if (response.success) {
  const message = response.message
  const data = response.data
}

// ❌ 错误写法
if (response.data.success) {  // 多了一层
  const data = response.data.data  // 多了一层
}
```

### 原因：响应拦截器

```javascript
// src/api/request.js
request.interceptors.response.use(
  async (response) => {
    const { data } = response
    return data  // ← 直接返回 response.data
  }
)
```
