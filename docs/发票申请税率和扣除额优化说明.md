# 发票申请税率和扣除额优化说明

## 修改内容

### 1. 税率字段优化

**修改前：**
- 使用数字输入框
- 需要手动输入小数（如 0.06 表示 6%）
- 容易输入错误

**修改后：**
- 使用下拉选择框
- 提供 1%-13% 的标准税率选项
- 显示百分比格式，更直观
- 避免输入错误

**可选税率：**
- 1% (0.01)
- 2% (0.02)
- 3% (0.03)
- 4% (0.04)
- 5% (0.05)
- 6% (0.06)
- 7% (0.07)
- 8% (0.08)
- 9% (0.09)
- 10% (0.10)
- 11% (0.11)
- 12% (0.12)
- 13% (0.13)

### 2. 扣除额字段优化

**修改前：**
- 所有开票方式都可以填写扣除额
- 没有限制

**修改后：**
- 只有开票方式为"全额"或"差额"时才能填写
- 其他开票方式（如"无"）时，扣除额字段自动禁用
- 切换开票方式时，如果不是全额或差额，自动清空扣除额

**开票方式说明：**
- **全额**：需要填写扣除额 ✅
- **差额**：需要填写扣除额 ✅
- **无**：不需要填写扣除额，字段禁用 ❌

## 使用说明

### 选择税率

1. 在发票详情表单中找到"税率"字段
2. 点击下拉框
3. 从列表中选择合适的税率（1%-13%）
4. 系统自动保存选择的税率值

### 填写扣除额

1. 首先选择"开票方式"
2. 如果选择"全额"或"差额"：
   - 扣除额字段可以填写
   - 输入具体的扣除金额
3. 如果选择"无"或其他：
   - 扣除额字段自动禁用（灰色）
   - 显示提示："仅全额和差额需要填写"
   - 已填写的扣除额会自动清空为 0

## 技术实现

### 前端修改

**文件：** `src/views/InvoiceApplications/index.vue`

#### 1. 税率字段改为下拉选择

```vue
<!-- 税率 -->
<el-col :span="12">
  <el-form-item label="税率">
    <el-select
      v-model="invoiceDetailsForm.tax_rate"
      placeholder="请选择税率"
      style="width: 100%"
    >
      <el-option label="1%" :value="0.01" />
      <el-option label="2%" :value="0.02" />
      <!-- ... 其他选项 ... -->
      <el-option label="13%" :value="0.13" />
    </el-select>
  </el-form-item>
</el-col>
```

#### 2. 扣除额字段添加禁用控制

```vue
<!-- 扣除额 -->
<el-col :span="12">
  <el-form-item label="扣除额">
    <el-input-number
      v-model="invoiceDetailsForm.deduction_amount"
      :precision="2"
      :min="0"
      :controls="false"
      :disabled="!needsDeductionAmount"
      style="width: 100%"
      placeholder="仅全额和差额需要填写"
    />
  </el-form-item>
</el-col>
```

#### 3. 添加计算属性

```javascript
// 判断是否需要填写扣除额（只有全额和差额需要）
const needsDeductionAmount = computed(() => {
  const method = invoiceDetailsForm.invoice_method
  return method === 'full' || method === 'diff'
})
```

#### 4. 添加监听器

```javascript
// 监听开票方式变化，自动清空扣除额（如果不是全额或差额）
watch(
  () => invoiceDetailsForm.invoice_method,
  (newMethod) => {
    if (newMethod !== 'full' && newMethod !== 'diff') {
      invoiceDetailsForm.deduction_amount = 0
    }
  }
)
```

## 数据存储

### 税率值

- 前端显示：百分比格式（如 "6%"）
- 数据库存储：小数格式（如 0.06）
- 数据类型：DECIMAL(5,4)

### 扣除额值

- 前端显示：金额格式（如 "1000.00"）
- 数据库存储：金额格式（如 1000.00）
- 数据类型：DECIMAL(10,2)
- 当开票方式不是全额或差额时，自动设为 0

## 注意事项

1. **税率选择**
   - 只能从预设的 1%-13% 中选择
   - 不能手动输入其他值
   - 如需其他税率，需要修改代码添加选项

2. **扣除额填写**
   - 必须先选择开票方式
   - 只有全额和差额才能填写
   - 切换开票方式时会自动清空不适用的扣除额

3. **数据验证**
   - 税率：必须选择一个值
   - 扣除额：最小值为 0，保留 2 位小数

4. **兼容性**
   - 已有数据不受影响
   - 新数据按新规则处理
   - 导出功能正常工作
