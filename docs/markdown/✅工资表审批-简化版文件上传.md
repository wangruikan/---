```vue
  <el-button type="primary" size="small">选择文件</el-button>

```

### 提交逻辑

```javascript
const handleConfirmSubmitApproval = async () => {
  // 1. 检查文件数量（必须大于0）
  if (fileList.value.length === 0) {
    ElMessage.warning('请先选择要上传的附件文件（至少1个）')
    return
  }

  // 2. 创建审批记录
  const response = await submitSalaryApproval({...})
  const approvalId = response.data.id

  // 3. 循环上传所有文件

  }

  // 4. 完成

}
```

## ✨ 特性

### 1. 必须上传文件
- 提交按钮在未选择文件时为禁用状态
- 点击提交时会检查文件数量
- 至少需要上传1个文件

### 2. 文件大小检查
- 选择文件时立即检查大小
- 超过50MB的文件会被自动拒绝
- 提示用户文件过大

### 3. 批量上传
- 可以一次选择多个文件（最多5个）
- 提交时自动循环上传所有文件
- 显示上传进度提示

### 4. 智能反馈
- 显示上传成功的文件数量
- 如果部分文件上传失败，会提示具体哪个文件失败
- 即使部分失败，成功的文件也会保存

## 📋 使用说明

### 提交工资表审批

1. **打开提交审批对话框**
   - 在工资表列表中点击"提交审批"

2. **填写审批信息**
   - 选择审批方式：线上审批 或 线下审批（盖章）
   - 填写备注信息（可选）

3. **选择要上传的文件**
   - 点击"选择文件"按钮
   - 可以一次选择多个文件
   - 支持的格式：PDF、Word、Excel、图片等
   - 单个文件最大50MB
   - 最多5个文件

4. **提交审批**
   - 确认已选择文件（至少1个）
   - 点击"提交审批"按钮
   - 等待上传完成
   - 看到"提交成功"提示

---

## ⚠️ 注意事项

1. **必须上传文件**
   - 至少选择1个文件才能提交
   - "提交审批"按钮在未选择文件时为禁用状态

2. **文件大小限制**
   - 单个文件不能超过50MB
   - 超大文件会被自动拒绝

3. **文件数量限制**
   - 最多可以上传5个文件
   - 超过限制无法继续添加

4. **上传失败处理**
   - 如果某个文件上传失败，其他成功的文件仍会保存
   - 会显示具体的成功/失败数量
