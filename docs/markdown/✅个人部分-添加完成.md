**说明**：
- 使用绿色 (`#67C23A`) 显示个人合计
- 加粗显示，便于区分

### 3. 动态标签方法

```javascript
// 获取当前月份标签（用于表头）
const getCurrentMonthLabel = (type = '单位') => {
  if (!currentSheet.value || !currentSheet.value.month) return `(本月)${type}部分`
  
  const month = currentSheet.value.month // 格式：2025-10
  const monthNumber = month.split('-')[1] // 提取月份：10
  
  return `(${parseInt(monthNumber)}月)${type}部分`
}
```

**用法**：
- `getCurrentMonthLabel()` → "(10月)单位部分"
- `getCurrentMonthLabel('单位')` → "(10月)单位部分"
- `getCurrentMonthLabel('个人')` → "(10月)个人部分"

## 🎯 单位部分 vs 个人部分

### 单位部分
```
(10月)单位部分
├─ 养老保险 (单位金额)
├─ 医疗保险 (单位金额)
├─ 失业保险 (单位金额)
├─ 公积金 (单位金额)
├─ 大额医疗 (单位金额)
├─ 社保基数 ✅ 有基数
├─ 公积金基数 ✅ 有基数
├─ 医保基数 ✅ 有基数
└─ 大额基数 ✅ 有基数
```

### 个人部分
```
(10月)个人部分
├─ 养老保险 (个人金额)
├─ 医疗保险 (个人金额)
├─ 失业保险 (个人金额)
├─ 公积金 (个人金额)
└─ 大额医疗 (个人金额)
❌ 无基数列
```

**区别**：
- ✅ 单位部分：显示单位金额 + 各类基数
- ✅ 个人部分：只显示个人金额，不显示基数
