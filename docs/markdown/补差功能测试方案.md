## 测试注意事项

1. **数据准备**：按照每个方案创建员工和调基记录
2. **上下限配置**：在相应的配置表中设置新旧上下限
3. **生效日期**：确保生效日期符合测试场景
4. **运行命令**：`php artisan base:process-compensation`
5. **验证结果**：
   - 检查 `insurance_compensation_records` 表
   - 确认补差期间、月数、金额是否正确
   - 验证约束逻辑是否生效
