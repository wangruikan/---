# 🎯 工资表审批文件上传功能 - 完整说明

## 📋 功能概述

工资表提交审批时必须上传附件（如盖章的工资表PDF、Excel文件等），审批人可以查看和下载这些附件。

---

## 🗄️ 数据库变更

### 执行SQL脚本

文件名：`create_salary_approval_attachments_table.sql`

```sql
CREATE TABLE `salary_approval_attachments` (
  `id` bigint(20) UNSIGNED NOT NULL AUTO_INCREMENT,
  `salary_approval_id` bigint(20) UNSIGNED NOT NULL COMMENT '工资表审批ID',
  `filename` varchar(255) NOT NULL COMMENT '原始文件名',
  `file_path` varchar(500) NOT NULL COMMENT '文件路径',
  `file_size` bigint(20) UNSIGNED NOT NULL COMMENT '文件大小（字节）',
  `mime_type` varchar(100) DEFAULT NULL COMMENT '文件类型',
  `uploaded_by` bigint(20) UNSIGNED NOT NULL COMMENT '上传人ID',
  `created_at` timestamp NULL DEFAULT NULL,
  `updated_at` timestamp NULL DEFAULT NULL,
  PRIMARY KEY (`id`),
  KEY `idx_salary_approval_id` (`salary_approval_id`),
  KEY `idx_uploaded_by` (`uploaded_by`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='工资表审批附件';
```

## 🔄 完整的审批流程（含文件上传）

### 1. 提交工资表审批

```
工资表列表 → 点击"提交审批" 
           ↓
  弹出审批对话框
  - 填写审批方式（线上/线下）

           ↓
  点击"提交审批"
  - 检查是否上传了附件
  - 提交成功
```

### 2. 审批人查看附件

```
审批管理 → 工资表审批列表 
        ↓
  点击查看详情
  - 显示附件列表
  - 可以下载附件
        ↓
  审批通过/拒绝
```

```

- id: 附件ID

- mime_type: 文件类型 (如: application/pdf)
```

## 🎯 使用说明

### 提交审批并上传附件

1. **进入工资管理页面**
   - 找到需要审批的工资表

2. **点击"提交审批"**
   - 填写审批方式（线上/线下）
   - 填写备注信息

3. **创建审批记录**
   - 点击"创建审批记录"按钮
   - 系统创建审批记录并返回ID

4. **上传附件**
   - 点击"选择文件"
   - 选择要上传的文件（支持多个）
   - 文件自动上传

5. **提交审批**
   - 确认已上传附件
   - 点击"提交审批"按钮

### 审批人查看附件

1. **进入审批管理**
   - 查看待审批的工资表

2. **点击查看详情**
   - 查看附件列表
   - 点击附件名称下载查看

3. **审批处理**
   - 根据附件内容进行审批
   - 选择通过或拒绝

## 🔍 API接口文档

### 1. 上传附件

```
POST /api/salary-approvals/attachments/upload

Headers:
  Authorization: Bearer {token}
  X-Account-Set-ID: {account_set_id}
  Content-Type: multipart/form-data

Body:
  salary_approval_id: 审批ID
  file: 文件

Response:
{
  "success": true,
  "message": "附件上传成功",
  "data": {
    "id": 1,
    "filename": "工资表-2025-10.pdf",
    "file_path": "salary_approvals/1/1730123456_abc123.pdf",
    "file_size": 1024000,
    "mime_type": "application/pdf",
    "uploaded_by": 1,
    "uploader": {
      "id": 1,
      "name": "张三"
    }
  }
}
```

### 2. 删除附件

```
DELETE /api/salary-approvals/attachments

Params:
  id: 附件ID

Response:
{
  "success": true,
  "message": "附件删除成功"
}
```

### 3. 获取附件列表

```
GET /api/salary-approvals/attachments

Params:
```

```
GET /api/salary-approvals/attachments/{id}/download

```

## ⚠️ 注意事项

1. **必须上传附件**
   - 提交审批时，附件为必填项
   - 至少上传1个文件

2. **只有待审批状态可以管理附件**
   - 已审批的记录不能再上传或删除附件

3. **权限控制**
   - 只有上传人或管理员可以删除附件
   - 所有人都可以查看和下载附件

4. **文件存储**
   - 文件保存在 `public/salary_approvals/` 目录
   - 确保该目录有写入权限

5. **删除附件**
   - 删除附件记录时，会同时删除物理文件
