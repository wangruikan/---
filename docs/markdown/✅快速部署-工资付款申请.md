# 工资付款申请功能 - 快速部署指南

## 📋 部署步骤

### 1️⃣ 执行数据库脚本

在数据库管理工具（如 Navicat、phpMyAdmin）中执行以下SQL文件：

```
✅工资付款申请-数据库脚本.sql
```

**或者使用命令行：**

```bash
mysql -u your_username -p your_database < ✅工资付款申请-数据库脚本.sql
```

### 2️⃣ 清除Laravel缓存

```bash
php artisan cache:clear
php artisan config:clear
php artisan route:clear
```

### 3️⃣ 验证目录权限

确保以下目录存在并有写入权限：

```bash
public/payment_requests/
```

**Windows PowerShell：**
```powershell
New-Item -Path "public\payment_requests" -ItemType Directory -Force
```

**Linux/Mac：**
```bash
mkdir -p public/payment_requests
chmod 755 public/payment_requests
```

### 4️⃣ 重启服务（如需要）

如果使用队列或Supervisor：

```bash
# 队列
php artisan queue:restart

# Supervisor
sudo supervisorctl restart all
```

## ✅ 功能验证

### 测试流程

1. **生成工资表**
   - 进入"工资管理"页面
   - 点击"生成工资表"
   - 选择项目和月份

2. **提交工资表审批**
   - 点击"提交审批"
   - 选择盖章方式（线上/线下）
   - 上传附件（至少1个文件）
   - 提交审批

3. **审批工资表**
   - 进入"审批管理 > 待办"
   - 找到工资表审批记录
   - 审批通过

4. **发起付款申请**
   - 回到"工资管理"页面
   - 审批通过的工资表显示"发起付款"按钮
   - 点击"发起付款"
   - 填写备注，上传附件
   - 提交付款申请

5. **审批付款申请**
   - 进入"审批管理 > 待办"
   - 找到工资付款申请记录
   - 审批通过

6. **查看付款申请**
   - 进入"付款申请"模块
   - 查看工资付款申请记录

## 🔍 常见问题排查

### 问题1: 发起付款按钮不显示

**检查项：**
- 工资表是否已审批通过？
- 前端是否正确获取 `salary_approval_id`？
- 浏览器控制台是否有错误？

**解决方案：**
```bash
# 清除缓存
php artisan cache:clear

# 检查数据库
SELECT * FROM salary_approvals WHERE status = 'approved';
```

### 问题2: 附件上传失败

**检查项：**
- `public/payment_requests` 目录是否存在？
- 目录是否有写入权限？
- 文件大小是否超过限制（50MB）？

**解决方案：**
```bash
# 检查目录
ls -la public/payment_requests

# 设置权限
chmod 755 public/payment_requests
```

### 问题3: 审批流程未创建

**检查项：**
- `approval_instance_id` 字段是否已添加？
- 审批流程配置是否正确？
- Laravel日志是否有错误？

**解决方案：**
```bash
# 检查日志
tail -f storage/logs/laravel.log

# 检查数据库字段
DESCRIBE payment_requests;
```

### 问题4: 付款申请列表为空

**检查项：**
- 是否已发起付款申请？
- 账套过滤是否正确？
- `payment_type` 字段值是否为 `'salary'`？

**解决方案：**
```sql
-- 检查数据
SELECT * FROM payment_requests WHERE payment_type = 'salary';
```
